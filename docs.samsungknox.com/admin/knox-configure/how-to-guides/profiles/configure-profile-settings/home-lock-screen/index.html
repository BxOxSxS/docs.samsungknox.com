Home & Lock screen

Last updated October 7th, 2025

These settings allow you to customize the appearance and functionality of the device’s home and lock screens.

The following settings are available:

  • Theme & wallpaper
    • Device
    • Theme
    • Wallpaper
  • Home & Lock screen
    • Device type
    • Device screen preview
    • Lock screen

Theme & wallpaper

You can add custom themes and wallpapers to customize your devices with your company’s unique branding.

Device

Select your profile’s target device type to view the available configuration options.

Theme

A Theme is a file that allows users to customize and personalize the visual appearance of their Galaxy device’s user interface, including elements like wallpapers, icons, fonts, and color schemes.

To add a theme to your profile, select your target Device’s OS, then upload your Theme APK file.

Themes are not available if you selected Tablet in the device selection list above.

Wallpaper

You can add a custom wallpaper on the home and lock screens of your devices. If you’re configuring a profile for Z Flip or Z Fold devices, you can also set different wallpapers for the main display and cover screen separately.

After you set a Default lock screen wallpaper or a Cover screen wallpaper, you’ll also get an additional option to change the font color of the Time/Date widget.

(Dynamic edition only) Select Prevent end users from changing the Home screen and Lock screen wallpapers if you also want to restrict the device user’s ability to customize the device wallpapers.

Home & Lock screen

You can configure how apps, widgets, and notifications appear on the device home and lock screens after enrollment.

Device type

Select your profile’s target Device type to view the available configuration options. After selecting the device type, you can also select the following options:

  • Customize favorite applications — Add a row of pinned app icons at the bottom of the screen. Once added, you can click the Add icon to select the apps to pin. For Galaxy Z Fold devices only, you can also pin favorite apps to the cover screen

  • Clear all favorite applications from the Home screen — Remove all favorite apps from the device home screen. For Galaxy Z Fold devices only, you can also choose to remove favorite apps from both the internal display and from the cover screen separately.

  • Clear all shortcuts from the Home screen — Remove all current app icons and widgets from the home screen. For Galaxy Z Fold devices only, you can also choose to remove all shortcuts from both the internal screen and from the cover screen separately.

Device screen preview

In this section, you can configure the layout of the Home and Lock screen panels. For Galaxy Z Fold devices, you can also configure the layout of the cover screen panels. Note that any applications, widgets, or content added as shortcuts to the Home screen will be downloaded during the configuration process (if required). This may increase the configuration time.

Since device layouts may vary, test this profile on a sample device for an accurate preview before deploying to production devices.

To configure the Home (or Cover) screen layout, do the following:

  • Select Lock home screen layout if you want to prevent users from modifying the placement of apps and widgets on their device home screens.

  • Under Select internal display grid, specify your Home screen layout dimensions.

    • For Galaxy Z Fold devices, specify your dimensions under Select cover display grid.

  • Click any Add icon in the preview image to add an app (either as a local APK or from Google Play), folder, content, or widget to the home screen.

  • Click Add another screen to add additional shortcuts as required. You can add up to six additional screens.

Google Play apps added as home screen shortcuts aren’t installed on the device during enrollment. Instead, when the device user taps the app shortcut, the app’s Play Store opens, allowing them to download the app.

Lock screen

Under LOCK SCREEN, you can customize the shortcuts and widgets that appear on the device when the screen is locked.

  • You can customize the shortcuts displayed on the bottom left and right corners of the device Lock screen. By default, these shortcuts are set up to point to the default phone and camera applications. To customize a shortcut, enter the app package name in the Left shortcut or Right shortcut fields.

  • You can hide specific widgets on the Lock screen by selecting them in the Hide widgets list. The following widgets can be hidden on the device: Time, Date, Owner information, Help Text, and Battery information widgets.

  • (Dynamic edition only) If you select Set automatic lock time, you can specify the time period between when a screen timeout occurs (set in Sound & display), and when the device locks.

  • You can select HOME SCREEN to display additional app notification options. This setting is only available to devices running Knox version 3.4 or higher.

    • Set Show notifications on the Home screen to On if you want users to view app notifications when they tap and hold an app shortcut on the Home screen.

    • Set to Off if you don’t want notifications showing when users tap and hold the app shortcut.
