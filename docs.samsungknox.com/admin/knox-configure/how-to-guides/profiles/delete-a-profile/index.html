Delete a profile

Last updated July 26th, 2023

Knox Configure profiles that aren’t assigned to any devices can be deleted at any time.

However, if you want to delete a profile that is assigned to a device, you’ll need to unassign the device or assign a different profile to the device before the profile can be deleted.

To delete a profile:

  1. On the Knox Configure console, go to Profiles.

  2. Select one or more target profiles to delete from the list of profiles.

  3. Click ACTIONS, then Delete profile(s).

  4. When prompted, review and confirm the deletion. If the profile was selected for auto-assignment for reseller uploaded devices, you’ll be warned that you’re deleting a default profile.
