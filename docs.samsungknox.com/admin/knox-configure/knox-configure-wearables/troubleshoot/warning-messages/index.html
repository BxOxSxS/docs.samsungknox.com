Warning messages

Last updated July 26th, 2023

You may encounter the following warning messages when using the Knox Configure console, this section of the guide provides troubleshooting tips for these messages.

Code Warning message Cause of Error Troubleshoot
91001 Warning: Unable to create shortcut for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Unable to create a shortcut for the specified app. The package name wasn't found. Verify that the app had been successfully installed.If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91002 Warning: Unable to change application icon(s) for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Can't change the app icon for the specified app. The package name wasn't found. Verify that the app had been successfully installed.If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91003 Warning: Unable to change application name(s) for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Can't change the specified app name. The package name wasn't found. Verify that the app had been successfully installed.If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91004 Warning: Unable to disable application(s) for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Can't disable the specified app. The package name wasn't found. Verify that the app had been successfully installed. If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91005 Warning: Unable to prevent application(s) for <package name1>, <package name2> from being uninstalled as the application is not installed. Check the package name(s) in your profile. Unable to prevent the specified app from being uninstalled. The package name wasn't found. Verify that the app had been successfully installed. If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91006 Warning: Unable to allow appear on top for<package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Unable to add an "appear on top" permission for the specified app. The package name wasn't found. Verify that the app had been successfully installed. If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91007 Warning: Unable to allow access data usage for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Unable to add a "Usage data access" permission for the specified app.The package name wasn't found. Verify that the app had been successfully installed.If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91008 Warning: Unable to allow changing system setting for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Unable to add a "Change system setting" permission for the specified app. The package name wasn't found. Verify that the app had been successfully installed. If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91009 Warning: Unable to restrict application(s) to doze mode for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile. Unable to add a "Application to prevent doze" permission for the specified app.The package name wasn't found. Verify that the app had been successfully installed.If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91010

Warning: Unable to add dex application shortcut(s) for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile.

 Unable to create a DeX shortcut for the specified app. The package name wasn't found. Verify that the app had been successfully installed. If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91011

Warning: Unable to disable dex application for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile.

 Can't disable the specified DeX app. The package name wasn't found Verify that the app had been successfully installed. If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
91012

Warning: Unable to remap hardware keys for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile.

 Unable to remap hardware keys for the specified app. The package name wasn't found. Verify that the app has been successfully installed. If you continue to experience issues, check the package name(s) and component name in your profile and verify that the names have been entered correctly.
91016

Warning: Unable to launch application(s) for <package name1>, <package name2> as application(s) is/are not installed. Check the package name(s) in your profile.

 Unable to launch the specified app. The package name is not installed. Verify that the app is successfully installed. If you continue to experience issues, check the package name(s) in your KC profile and verify that the package name is entered correctly.
92001

Warning: Unable to create shortcut(s) for <package name1>, <package name2> as application(s) is/are in disabled mode. Check the package name(s) in your profile.

 Can't disable the specified app. The app has been disabled. Verify that the app hasn't been disabled. If the app has been disabled, the app must be enabled before a shortcut can be added.
92002

Warning: Unable to change application icon(s) for <package name1>, <package name2> as application(s) is/are in disabled mode. Check the package name(s) in your profile.

 Can't change the app icon for the specified app.The app has been disabled. Verify that the app hasn't been disabled. If the app has been disabled, the app must be enabled before the app icon can be modified.
92003

Warning: Unable to change application name(s) for <package name1>, <package name2> as the application(s) is/are in disbled mode. Check the package name(s) in your profile.

 Can't change the specified app name.The app has been disabled. Verify that the app hasn't been disabled. If the app has been disabled, the app must be enabled before the app name can be modified.
92004

Warning: Unable to restrict application(s) in doze mode for <package name1>, <package name2> as the application(s) is/are in disbled mode. Check the package name(s) in your profile.

 Unable to add a "Application to prevent doze" permission for the specified app.The app has been disabled. Verify that the app hasn't been disabled. If the app has been disabled, the app must be enabled before app permissions can be modified.
92005

Warning: Unable to add dex shortcut application(s) for <package name1>, <package name2> as the application(s) is/are in disbled mode. Check the package name(s) in your profile.

 Unable to create a DeX shortcut for the specified app. The package name has been disabled. Verify that the app hasn't been disabled. If the app has been disabled, the app must be enabled before creating a shortcut.
92006

Warning: Unable to disable dex application(s) for <package name1>, <package name2> as the application(s) is/are in disabled mode. Check the package name(s) in your profile.

 Unable to disable the specified DeX app. The package name has already been disabled. Verify that the app hasn't already been disabled.
92012

Warning: Unable to launch application(s) for <package name1>, <package name2> as application(s) is/are disabled. Check the package name(s) in your profile.

 Unable to launch the specified app. The package name is disabled. Verify that the app had been successfully installed and enabled. If you continue to experience issues, check the package name(s) in your profile and verify that the package name has been entered correctly.
93100 Warning: Unable to disable OMC mode as OMC functionality is not supported in the target device. Check the device and your profile. Can't disable Open Market Customization (OMC) mode. The target device doesn't support this functionality. OMC mode must be supported on the target device before modifying the OMC settings. Check the device profile to verify compatibility.
93200 Warning: Unable to enable SIM lock as SIM card functionality is not supported in the target device. Check the device and/or your profile. Can't enable SIM lock. The target device doesn't support this functionality. SIM lock must be supported on the target device before modifying the SIM lock settings. Check the device profile to verify compatibility.
93201 Warning: Unable to enable SIM 1 lock as SIM card is not inserted to SIM 1 slot in the target device. Check the device. Can't enable SIM lock on the SIM 1 slot. The target device doesn't have a SIM card inserted on the SIM 1 slot. Verify that a SIM card has been inserted on the SIM 1 slot before enabling SIM lock.
93202 Warning: Unable to enable SIM 2 lock as dual SIM card functionality is not supported in the target device. Check the device and/or your profile. Can't enable SIM lock on the SIM 2 slot. The target device doesn't have the dual SIM feature. SIM lock on the SIM 2 slot is only supported on devices with the dual SIM feature. Check the target device for compatibility.
93203 Warning: Unable to enable SIM 2 lock as SIM card is not inserted to SIM 2 slot in the target device. Check the device. Can't enable SIM lock on the SIM 2 slot. The target device doesn't have a SIM card inserted on the SIM 2 slot. If the device has the Dual SIM feature, verify that a SIM card has been inserted on the SIM 2 slot before enabling SIM lock.
93300 Warning: Unable to disable NFC as NFC feature is not supported in the target device. Check the device and/or your profile. Can't enable NFC. The target device doesn't support NFC. NFC can only be enabled on supported devices. Check the target device for compatibility.
93301 Warning: Unable to change NFC setting as NFC feature is not supported in the target device. Check the device and/or your profile. Can't modify NFC settings. The target device doesn't support NFC. NFC can only be modified on supported devices. Check the target device for compatibility.
93302 Warning: Unable to prevent from changing NFC setting as NFC feature is not supported in the target device. Check the device and/or your profile. Can't prevent change settings for NFC. The target device doesn't support NFC. You can only prevent the settings from being modified for devices that support NFC. Check the target device for compatibility.
93303 Warning: Unable to set ringtone as feature is not supported in the target device. Check the device and/or your profile. Can't set the ringtone on the device. The target device doesn't support the ringtone feature. You can only set the ringtone for devices that support this feature. Check the target device for compatibility.
93312 Warning: Unable to enable battery protection as feature is not supported in the target device. Check the device and/or your profile. The battery setting is not supported on this device. Check that you are on the correct device. This feature will work for tablets, Galaxy XCover Pro/XCover5 with R OS only, and phones from S OS.
93325 Warning: Unable to open files <file title1>, <file title2>, ... because there is no available application that can open them. The WebKiosk app is not available on the device. Install or enable at least one (1) app capable of opening the target file on the device.
94400

Warning: Unable to create shortcut(s) for <package name1\activity name1>, <package name1\activity name2>, <package name2\activity name1> as activity of application(s) is/are not present. Check the activity name(s) in your profile.

 Unable to add shortcut for the specified activity. The application activity wasn't found. Verify that the app had been successfully installed.If you continue to experience issues, check the package name(s) and activity name in your profile and verify that the names has been entered correctly.
94401

Warning: Unable to disable <package name1\component name 1>, <package name1\component name 2>, <package name2\component name3> as activity/component of application(s) is/are not present. Check the activity name(s) in your profile.

 Can't disable an invalid activity/component. The <package name1\component name1> wasn't found. Verify that the app had been successfully installed.If you continue to experience issues, check the package name(s) and component name in your profile and verify that the names has been entered correctly.
94401

Warning: Unable to disable pre-installed browsers for <package name1>, <package name2>, ... as the application(s) may be disabled by some other application.

 Unable to disable pre-installed browsers. The package is disabled by another app. Check disabled features in other management apps such as Family Link parent app.
95002

Warning: Unable to add application URL restrictions for <package name1>, <package name2>, ... as the application(s) may be disabled by some other application.

 Unable to add URL restrictions. The package is disabled by another app. Check disabled features in other management apps such as Family Link parent app.
