If a trial license expires before it’s upgraded, how can I apply a commercial license?

Last updated July 26th, 2023

Environment

  • Knox Configure

Overview

If a trial license has expired, you must assign devices with a new valid license. After you assign devices in the Knox Configure console, on the device the user must go to Settings > About phone > Software info > Knox Configure > Apply latest profile to trigger re-enrollment in Knox Configure. Additionally, devices can be re-enrolled after a factory reset or at configure.samsungknox.com.
