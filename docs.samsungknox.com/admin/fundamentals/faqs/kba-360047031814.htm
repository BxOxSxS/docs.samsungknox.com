Cannot invite administrator to Knox Cloud Services

Environment

  • Knox Cloud Services (KCS)

Overview

When inviting an administrator to a KCS solution, the following error is shown:

[user email] has already been invited by a different customer. Please ask the user to contact Samsung Knox Customer Support if this is an error.

Cause

An email can only be invited as an administrator to a Knox Cloud Services (KCS) once, and can only be associated with one company account.

Resolution

In your KCS solution's console, on the Administrators & roles page, check that the user has not already been invited to the company.

NOTE—If you have signed up for multiple KCS solutions, please check the Administrators & roles pages of each service individually.

NOTE—If you are using KME, on the Administrators & roles page, ensure Show blocked and revoked administrators is selected.

If the email invite shows as Pending, Blocked or Revoked, resend the invitation:

  1. In Administrators & roles, select the invited user.
  2. Select Edit administrator.
  3. In Edit Administrator, select Resend invitation.

If the user's status is Blocked or Revoked, this should change the status to Pending. The status updates to Active when the user activates their account by following the invitation link sent to their email.

If the user cannot be found in Administrators & roles, please submit a ticket through your Knox dashboard. Ensure you include the information below, along with any account access issues.

The invited user's email

The customer ID you are trying to add the user to

Additional information

To learn more about administrators, see the respective admin guide for your KCS solution
