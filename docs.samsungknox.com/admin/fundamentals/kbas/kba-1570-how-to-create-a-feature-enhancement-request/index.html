How to create a feature enhancement request

Last updated June 4th, 2026

Categories:

Account

Environment

  • Knox cloud services

Overview

When using Samsung Knox cloud services, you may want to request a new feature or an improvement to an existing feature. You can create a feature enhancement request (FER) for your suggestions.

The Samsung Knox team reviews all feature requests, and upon review may accept or reject an FER.

How to create a feature enhancement request

You can create a FER using one of the following methods, depending on your Knox service plan and support level.

For Knox Mobile Enrollment and Knox Configure customers

You can submit FERs directly through Knox Admin Portal:

  1. Sign in to Knox Admin Portal.
  2. Click (Help icon) next to your account icon.
  3. Select Send feedback and complete the form with your FER details.
  4. Click SUBMIT.

For Enterprise Technical Support customers

If you have Enterprise Technical Support, use the following template to create and submit a support ticket:

Subject: [FER] Your title summary

Description (please include the following fields in the ticket description):

  • Problem or pain point: Please provide a detailed description of the pain points you see in your solution.
  • Expected function and result of solution: Please describe in detail what you would like the solution to do for your business.
  • Customer use case: Please provide the specific use case that requires this feature or enhancement.
  • Business opportunity information: Please provide your current license seat quantity or future rollout plan.
  • Additional information (optional): Please provide any related images, videos, device screenshots, etc. of the pain point.
