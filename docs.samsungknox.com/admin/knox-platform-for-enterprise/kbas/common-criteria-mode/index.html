Common Criteria Mode

Last updated July 26th, 2023

Categories:

Security

Download the Common Criteria Mode APK file and refer to the latest list of certified devices for NIAP certification.

  1. Android apk which supports enabling CC mode on a device. Intended for NIAP evaluators or IT admins who want to test the CC mode. CC mode should be turned on/off only by the EMM policy.
    • For Android Marshmallow (6.x) or below: CCMode.apk v1.1
    • For Android Nougat (7.x): CCMode.apk v1.2
    • For Android Oreo (8.x): CCMode.apk v2.1
    • For Android Pie (9.x): CCMode.apk v3.0
    • For Android 10 (10.x): CCMode.apk v3.1
    • For Android 11 (11.x) and Android 12 (12.x): CCMode.apk v3.2
  2. Guidance documents for CC evaluation
  3. Application List: Contains version information of each certified device
