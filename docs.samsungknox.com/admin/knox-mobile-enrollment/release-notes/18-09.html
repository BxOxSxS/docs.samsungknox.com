Knox Mobile Enrollment 1.15 release notes

Last updated July 26th, 2023

Knox Mobile Enrollment (KME)

BlackBerry MDM DO support

BlackBerry now has Google DO (Work Managed Device) support for KME.

Sophos Mobile MDM DO support

Sophos Mobile now has Google DO (Work Managed Device) support for KME.

Samsung SDS EMM DO support

Samsung SDS EMM now has Google DO (Work Managed Device) support for KME.
