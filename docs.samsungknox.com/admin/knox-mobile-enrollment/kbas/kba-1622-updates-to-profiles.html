Updates to profiles

Last updated July 18th, 2025

Categories:

Profile

Environment

  • Knox Mobile Enrollment 25.07
  • Knox E-FOTA 25.07
  • Knox Asset Intelligence 25.07

Overview

Starting with the Knox Mobile Enrollment 25.07 release, updates were made to Knox Mobile Enrollment profiles. The Knox Mobile Enrollment team has migrated existing profiles as enrollment profiles.

You can now view your enrollment profiles on the Enrollment page of the Knox Admin Portal.

For most admins, no further action is required. However, depending on the configuration for existing profiles in Knox Mobile Enrollment, Knox E-FOTA, and Knox Asset Intelligence, there may be some cases where action is required.

Case where action is required

If you have an existing Knox Mobile Enrollment profile with Auto approval and profile auto-assignment enabled, and if multiple resellers have different configurations for Knox E-FOTA Auto enrollment or Knox Asset Intelligence Auto enrollment under the same profile, action is required to create an enrollment profile for each reseller.

The following table details the conditions involved in the case where action is required:

Knox Mobile Enrollment Knox E-FOTA or Knox Asset Intelligence Behaviour

The same profile is assigned to both reseller A and B with the following configuration:

  • Auto approval enabled
  • Profile auto-assignment enabled

A reseller exists for Knox E-FOTA or Knox Asset Intelligence with the following preferences:

  • Auto enrollment enabled for reseller A
  • Auto enrollment disabled for reseller B
If the same profile is assigned to multiple resellers with different preferences, action is required to create a profile for each reseller. If any action is required, an email has been sent with further instructions from the Knox Mobile Enrollment team.

A profile exists with the following configuration:

  • Auto approval enabled
  • Profile auto-assignment enabled

A reseller exists for Knox E-FOTA or Knox Asset Intelligence with the following preferences:

  • Auto enrollment configuration is the same across reseller A and B
The profile and preferences are automatically migrated for reseller A and B. No further action is required.

Cases where no action is required

Your existing Knox Mobile Enrollment profile is automatically migrated to an enrollment profile for the following conditions:

Knox Mobile Enrollment Knox E-FOTA or Knox Asset Intelligence Behaviour

A profile exists with the following configuration:

  • Auto approval enabled or false
  • Profile auto-assignment enabled
No reseller exists for Knox E-FOTA or Knox Asset Intelligence

Your existing Knox Mobile Enrollment profile is automatically migrated to a new Enrollment profile with the following options selected:

  • EMM

A reseller exists for Knox E-FOTA with the following preferences:

  • Auto approval enabled
  • Auto enrollment enabled

Your existing Knox Mobile Enrollment profile is automatically migrated to a new Enrollment profile with the following options selected:

  • EMM
  • Knox E-FOTA

A reseller exists for Knox Asset Intelligence with the following preferences:

  • Auto approval enabled
  • Auto enrollment enabled

Your existing Knox Mobile Enrollment profile is automatically migrated to a new Enrollment profile with the following options selected:

  • EMM
  • Knox Asset Intelligence

A reseller exists for Knox E-FOTA and Knox Asset Intelligence with the following preferences:

  • Auto approval enabled
  • Auto enrollment enabled

Your existing Knox Mobile Enrollment profile is automatically migrated to a new Enrollment profile with the following options selected:

  • EMM
  • Knox E-FOTA
  • Knox Asset Intelligence

A new enrollment profile is automatically created for the following conditions:

Knox Mobile Enrollment Knox E-FOTA or Knox Asset Intelligence Behaviour

A profile either doesn't exist, or exists with the following configuration:

  • Auto approval disabled

A reseller exists for Knox E-FOTA with the following preferences:

  • Auto approval enabled
  • Auto enrollment enabled or disabled

A new enrollment profile is automatically created with the following name:

(reseller name)_(reseller ID)_reseller_enrollment_profile

The following options are selected for the new profile:

  • Knox E-FOTA

A reseller exists for Knox Asset Intelligence with the following preferences:

  • Auto approval enabled
  • Auto enrollment enabled or disabled

A new enrollment profile is automatically created with the following name:

(reseller name)_(reseller ID)_reseller_enrollment_profile

The following options are selected for the new profile:

  • Knox Asset Intelligence

A reseller exists for Knox E-FOTA and Knox Asset Intelligence with the following preferences:

  • Auto approval enabled
  • Auto enrollment enabled or disabled

A new enrollment profile is automatically created with the following name:

(reseller name)_(reseller ID)_reseller_enrollment_profile

The following options are selected for the new profile:

  • Knox E-FOTA
  • Knox Asset Intelligence
