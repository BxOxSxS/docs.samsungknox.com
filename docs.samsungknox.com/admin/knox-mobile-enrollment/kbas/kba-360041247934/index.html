Skip Setup Wizard not working when using Knox Configure and Knox Mobile Enrollment togethe

Last updated July 26th, 2023

Environment

  • Knox Mobile Enrollment (KME)
  • Knox version: 2.7.1
  • Devices assigned to a Knox Configure (KC) profile and Knox Mobile Enrollment (KME) profile

Overview

Some users are reporting that the “Skip Setup Wizard” feature is not working when the following conditions are met on the device:

  1. Running Knox version 2.7.1
  2. Enrolling into a KME Device Admin profile with Skip Setup Wizard selected
  3. Assigned a profile in KC

Cause

In KC, the “Skip Setup Wizard” feature is not supported until Knox version 2.8. As the enrollment process pulls down the KC profile first, the device complies with the setup preferences in the KC profile instead of the KME preferences.

Note

You can find your device’s Knox version by navigating to Settings > About Phone > Software Information > Knox Version.

Workaround

If Skip Setup Wizard is needed, ensure that the device is only assigned to the KME profile when enrolling. After the device is enrolled into KME, assign the device to KC and configure the device by navigating to configure.samsungknox.com and selecting “First-Time Enrollment'.
