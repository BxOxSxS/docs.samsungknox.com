Device details

Last updated March 20th, 2024

On this tab

  • Assign a profile to a device
  • Unassign a profile from a device
  • Delete a device
  • Delete devices in bulk

The Knox Mobile Enrollment console enables you to manage devices either individually or in bulk, and change the associated profiles as needed. You can also delete devices as necessary.

To view the details of an approved device:

  1. On the Devices page, click the All devices tab and review the device list. You can also click to show and hide columns in the list.

  2. This list displays all the following information:

    • IMEI/MEID — Represents the unique identifier of each device. Click the IMEI/MEID to view detailed information about the user ID and the associated device. Dual-SIM devices display only one of their IMEIs in the device list, but you can search for dual-SIM devices using either their first or second IMEI. If a device has two SIMs, both IMEIs are shown in its details.

      To search for a device by its IMEI, the substring you enter is acceptable only if it matches the start of the IMEI. For example, if the IMEI is 123456789, any substring that starts with 1 is a valid entry.

      Important

      See the updates to device search for more information on how you can you search for a device.

    • SERIAL NUMBER — Click the serial number to view the device’s details, such as its ID, model, status, profile, upload dates, and much more.

    • MODEL — Filter the device models displayed in the list.

    • USER ID — Use the user ID to determine the credentials you assign to a particular device user and review information about the device.

    • TAGS — Filter devices by their associated tags.

    • PROFILE — Review the profiles assigned to each device. See Android Enterprise profiles to learn how you can create an Android Enterprise profile.

    • STATUS — Monitor the enrollment status and the current profile of each listed device.

    • SUBMITTED — Determine when the reseller added the device.

    • LAST SEEN — Logs the most recent ping Knox Mobile Enrollment sent to the server.

  3. (Optional) Set the following device parameters:

    • User ID and Password — Click a device’s IMEI or serial number to change its user ID and password.

    • Tags — Enter any tags you’d like to assign to the devices to easily organize them.

  4. (Optional) Click Actions > Download devices as CSV to download all details of all uploaded and approved devices in one swift action.

Assign a profile to a device

You can assign your profile to one or more devices:

  1. Sign in to the Knox Admin Portal.

  2. Go to Devices.

  3. Select one or more devices to assign the profile to:

    • One device — Click Device IMEI and select the desired profile from the Profile list.

    • Multiple devices:

      1. Select devices to assign the profile to.

      2. Click Actions > Configure devices.

      3. Edit the profile of the selected devices. Additionally, you can add tags and edit user credentials.

      4. Click Save.

To upload user credentials for vast numbers of device users at a time, such as thousands, see how you can upload device user credentials in bulk.

Unassign a profile from a device

You can unassign the profile assigned to a device:

  1. On the Devices page, click a device’s identifier in the IMEI column.
  2. Select a profile from the Profile list.
  3. Select Clear profiles to clear a profile.
  4. Click Save.

You can also unassign a profile from multiple devices in one action:

  1. On the Devices page, select multiple device IMEIs.

  2. Click Actions > Configure devices. To use this action, you need the Assign with profile and manage tags permission. See Manage admin and roles for more information.

  3. Unassign the profile assigned to the selected devices by selecting Clear profiles in the Profiles list.

  4. Click Save.

Delete a device

Once you remove a device from your tenant, it is permanently removed from your tenant. See all the ways you can upload a device to your account. To do so, on the Devices page:

  1. Click All devices.
  2. Select one or more devices to delete.
  3. Click Actions > Delete devices.
  4. Click Delete.

Delete devices in bulk

You can also delete devices in bulk through a CSV file containing device IMEIs:

  1. Open Bulk actions tab. Click View instructions to learn how to format the device IMEI in the CSV file.
  2. Click Bulk delete.
  3. Upload the CSV file and click Submit.
  4. Click Delete to finish the deletion.
