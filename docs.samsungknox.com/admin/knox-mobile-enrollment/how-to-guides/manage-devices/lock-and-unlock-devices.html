Lock and unlock devices

Last updated August 23rd, 2023

On this tab

  • Lock devices
  • Check the device lock status
  • Lock and unlock switch
  • Unlock devices using a PIN
  • Recovering Google FRP locked devices

Knox Mobile Enrollment advanced profiles let you lock and unlock one or more devices. With the ability to lock lost or stolen devices and unlock devices in offline environments, advanced profiles help you protect your data in such adverse scenarios.This page explains how you can lock or unlock your devices with advanced profiles.

Lock devices

On the Advanced tab, you can simultaneously lock or unlock one or more devices:

  1. Select the devices you want to lock or unlock.

  2. Click Actions.

  3. Lock the switch if you want to lock devices, or Unlock if you want to unlock them. The device status updates immediately.

Check the device lock status

On the Advanced tab, there are four possible device lock statuses:

  • Locked — The device is locked.

  • Unlocked — The device is unlocked.

  • Locking — The device is in the process of locking.

  • Unlocking — The device is in the process of unlocking.

Lock and unlock switch

The device lock controls are color-coded to relay status information to you at a glance:

StatusSwitch labelSwitch color
UnlockedUnlocked label is boldGreen
Locking (or Unlocking)No label is boldYellow
LockedLocked label is boldRed

Unlock devices using a PIN

Devices that are assigned advanced profiles can be unlocked in highly secure offline environments using a one-time PIN generated by Knox Mobile Enrollment:

  1. If the device profile status is Unenrolled, and the device lock status is Locked, on the Devices page, select a device IMEI and click PIN.

  2. On the Unlock device dialog, enter the 8-digit code that is displayed on the device user’s lock screen.

  3. Click Continue to generate a one-time 8-digit PIN.

  4. Share the generated PIN with the device user. The device user uses this PIN to unlock their device.

Recovering Google FRP locked devices

Factory reset protection (FRP) allows only a device user with Google credentials to factory reset and activate their device. This feature is auto-enabled on devices running Android 5 or higher. Hence, unauthorized individuals won’t be able to reset such a device, protecting device information even when the device is lost or stolen.

Additionally, as enterprise devices might get locked with an employee’s Google account, it may cause inconvenience when the employee leaves the company — the device may become unusable as a company asset. You can recover such devices by bypassing the factory reset protection.

Note

An FRP locked device can be unlocked when you select Skip setup wizard while creating an Android Legacy profile.

To bypass factory reset protection:

  1. Ensure that the device is assigned a Knox Mobile Enrollment profile with the following options:

  2. Select Skip setup wizard. This setting is selected by default only for Android Legacy (Device Admin) profiles, but not for Android Enterprise and Android Enterprise (Advanced) profiles. For more information, see Android Enterprise profiles.

  3. Select Allow end user to cancel enrollment. This setting isn’t selected by default for Android Legacy (Device Admin) profiles. Also, this feature isn’t supported by Android Enterprise and Android Enterprise (Advanced) profiles. For more information, see Android Enterprise profiles.

  4. Factory reset the device.

  5. After the device powers on, connect to a Wi-Fi network. You’ll be prompted for a reboot.

  6. Reboot the device. The enrollment proceeds without prompting you for Google login credentials.
