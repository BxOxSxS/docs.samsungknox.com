Manage devices in bulk

Last updated August 5th, 2025

Knox Mobile Enrollment makes it easy to configure thousands of devices at the same time using a properly formatted CSV file. Simply download your device IMEIs as a CSV file, edit the data in a spreadsheet, then upload the CSV file to your console.

This page walks you through how to format your CSV files in order to perform device operations in bulk. Alternatively, you can click View instructions on the BULK ACTIONS page for an in-console explanation of how to format the CSV file.

Bulk actions are also available through the Knox Admin Portal’s common device list. For more information, see Bulk actions.

Assign enrollment profile

To assign user credentials and an enrollment profile to up to 100,000 devices:

  1. On the Devices page, select the devices you want to configure.

  2. Click ACTIONS > Download devices as CSV.

  3. Open the data in the CSV file in any program that stores data in tables, such as Microsoft Excel or Google Sheets.

  4. Configure your spreadsheet as follows:

    • Column A — Remove the header row and leave pre-populated device IMEIs.
    • Column B — The profile name. You can assign different profiles in the same file, as long as you specify only one profile per device.
    • Column C — (Optional) Enter a User ID.
    • Column D — (Optional) Enter a password.

    Your spreadsheet should look something like this:

    Column A Column B Column C Column D
    Field Device IMEI or serial number Profile User ID Password
    Example 350925370583126 Profile name A alex_doe@example.com QSu7hdD5,

  5. Remove the rest of the content from the other columns and save the file as a CSV file.

  6. Go to Devices > BULK ACTIONS.

  7. Select Assign enrollment profile.

  8. Upload the CSV file containing device IMEIs and user credentials.

  9. (Optional) Select Overwrite user credentials to replace the existing user credentials with credentials from the CSV file.

  10. Click Submit to upload the file to your console.

Unassign enrollment profile

To unassign enrollment profiles from up to 100,000 devices:

  1. On the Devices page, select the devices you want to remove profiles from.

  2. Click ACTIONS > Download devices as CSV.

  3. Open the data in the CSV file in any program that stores data in tables, such as Microsoft Excel or Google Sheets.

  4. Configure your spreadsheet as follows:

    • Column A — Remove the header row and leave pre-populated device IMEIs. Then, remove the rest of the content from the other columns and save as a CSV file.

    Your spreadsheet should look something like this:

    Column A
    Field Device IMEI or serial number
    Example 350925370583126

  5. Go to Devices > BULK ACTIONS.

  6. Select Unassign enrollment profile.

  7. Upload the CSV file containing the device IMEIs.

  8. Click Submit to upload the file to your console.

Upload user credentials

To import user credentials for up to 100,000 users:

  1. Create a CSV file with one row (line) per device file in any program that stores data in tables, such as Microsoft Excel or Google Sheets.

  2. Configure your spreadsheet as follows:

    • Column A — Enter the User ID.
    • Column B — Enter a password.

    Your spreadsheet should look something like this:

    Column A Column B
    Field User ID Password
    Example alex_doe@example.com QSu7hdD5

  3. Go to Devices > BULK ACTIONS.

  4. Select Upload user credentials.

  5. Upload the CSV file.

  6. Click Submit to upload the file to your console.

Delete devices

You can delete up to 100,000 devices:

  1. On the Devices page, select the devices you want to delete.

  2. Click ACTIONS > Download devices as CSV.

  3. Open the data in the CSV file in any program that stores data in tables, such as Microsoft Excel or Google Sheets.

  4. Configure your spreadsheet as follows:

    • Column A — Remove the header row and leave pre-populated device IMEIs. Then, remove the rest of the content from the other columns and save the file as a CSV file.

    Your spreadsheet should look something like this:

    Column A
    Field Device IMEI or serial number
    Example 350925370583126

  5. Go to Devices > BULK ACTIONS.

  6. Select Bulk delete.

    Once you submit the CSV file, the devices are deleted from the Knox Mobile Enrollment device list. If the devices you delete are associated with another Knox service, such as Knox E-FOTA or Knox Configure, they’ll remain in the Knox Admin Portal’s common device list.

    If you delete a device that isn’t associated with any other Knox service, it’s permanently removed from the Knox Admin Portal. If you want the devices back, you’ll need to ask your reseller to upload them again.

  7. Upload the CSV file, then click Submit.
