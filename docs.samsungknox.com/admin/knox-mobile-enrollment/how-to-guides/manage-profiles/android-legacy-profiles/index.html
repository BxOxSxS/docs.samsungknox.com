Android Legacy profiles

Last updated August 18th, 2023

Knox Mobile Enrollment also supports employee-owned devices running Android 10 or lower through the Android Legacy (Device Admin) profiles.

This profile allows you to specify an EMM server URI to download specific EMM configurations, APK and custom JSON data for EMM setup. You can also skip the device setup flow and allow the device user to cancel enrollment.

To configure such a profile:

  1. On the Profiles page, click Create profile.

  2. Click the Android Legacy (Device Admin) profile link.

  3. Enter a profile name and an optional description.

  4. (Optional) Enter the EMM server URI for the profile. This URI points to the EMM’s installation portal for downloading specific EMM configurations. Verify you can connect to the EMM, since they may be firewall-protected or unavailable from public networks.

  5. Click Continue to set the Android Legacy profile settings.

  6. Set the EMM configuration settings:

    • EMM Agent APK — Add one or more EMM applications to be downloaded automatically upon device enrollment when the device first connects to Wi-Fi.

    • Custom JSON Data (as defined by EMM) — Provide a customized configuration type, in accordance with the JSON template your EMM expects.

  7. Configure the Device settings:

    • Skip Setup Wizard — Selecting this feature allows the device user to skip the wizard screens and finish the enrollment process in lesser steps. This setting is selected by default.

    • Allow end user to cancel enrollment — Permits the device user to cancel enrollment on their device. Unselecting this option enables mandatory device enrollment. Skip setup wizard functions independently from enrollment cancellation, and both can be enabled at the same time.

    • Privacy Policy, EULAs and Terms of Service — Display Knox related EULAs (License EULA) during initial enrollment to reduce the number of dialogs. In addition to default EULAs, any enterprise and EMM-specific EULAs can be added while creating a profile. Click the Samsung Knox Privacy Policy link to review the specific Privacy Policy text displayed to device users based on their geographic region. Optionally, click Add legal agreement to enter the Agreement Title and Agreement Text.

    • Support contact details — Select Add to update the Company Name, Company Address, Contact phone number, and Contact email address displayed on the device upon successful enrollment. If necessary, select Remember support contact details to use this information as the default contact information. If support for fully managed devices and work profile modes is enabled for the profile, then only the Customer Name is editable.

    • Associate a Knox license with this profile — Select this option to pass the Knox license key directly to the intended device for easier Knox profile configuration.

  8. Click Create to finish creating the profile.

The Profiles page now displays the newly created profile.
