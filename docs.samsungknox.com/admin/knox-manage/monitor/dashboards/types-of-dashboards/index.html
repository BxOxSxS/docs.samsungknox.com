Types of dashboards

Last updated July 22nd, 2025

A dashboard allows you to view important information at a glance about your devices, users, licenses, and more.

There are three basic types of dashboards:

  • Default dashboard
  • Device location
  • EMM dashboard

Default dashboard

Component Description
Device Shows the number of devices for each status. If issues exist, click the red button indicating the number of issues to see a list. Click an issue in the list to open a filtered device list displaying all devices with the selected issue.
User Shows the number of users. Click Request Enrollment to email a device enrollment guide to all users who have no enrolled devices.
Last Seen Shows the number of devices last seen per time period. Click Go to Device List to see a filtered device list of devices that were last seen 24 hours ago or earlier.
Device Command History Shows the number of device commands sent per day. If issues exist, click the red button indicating the number of issues to show the Requested Device Command link. Click the link to open the Device Command in Request page.
Assigned Profile Shows the top 10 profiles with the most assigned devices.
Assigned Application Shows the top 10 apps most assigned to devices.
Enrollment History Shows the number of devices enrolled over time (days). Click User to see number of users enrolled over time (days).
Platform Type Shows the count of enrolled devices by platform. Click a number to open a filtered list showing all devices for that platform.

Data collection for enrollment history begins 10 minutes after the default time zone is configured and may take up to two hours to complete. Viewing data during the collection process results in the count appearing as zero.
License (Knox Suite) Shows Knox Suite license consumption and remaining days for each license type. For a more detailed view, click See Detail to open the License page.

To view device locations on the Dashboard page:

  1. Click > Modify Dashboard. The Modify Default Dashboard dialog appears.
  2. In the Modify Default Dashboard dialog, select Use, then click OK.

Youll need to refresh the page if the Location widget doesnt appear initially. The widget displays up to 30,000 devices, sorted by most recent updates.

Device location

Displays the last known location and details of devices on Google Maps, excluding deactivated devices. Use the search field to find devices by user ID.

EMM dashboard

Component Description
New User Weekly Displays the users newly added within a week.
App Download Top3 (Current) Displays the three most downloaded apps up to the current date.
Device Count by Platform Displays the number and ratio of enrolled devices by mobile OS. You can exclude or include the item to display by clicking the chart legend.
Device Count by Status Displays the number and ratio of enrolled devices by status. You can exclude or include the item to display by clicking the chart legend.

Widget type dashboards

More widgets that can be selected for a widget type dashboard. The components of the default dashboard can be used as widgets.

Component Description
Assigned Content Displays up to ten of the most assigned content items.
Android Management Type Displays the number of Android devices by management type.
Users by Organization Displays the number of activated users by organization in descending order.
Group & Organization Displays the number of groups and organizations.
AD/LDAP Displays the number of updated devices through AD/LDAP sync.
Device Location Displays the last location of devices and their information on Google Maps. All devices are displayed except for unenrolled devices. Use the search field to search for a user ID.
