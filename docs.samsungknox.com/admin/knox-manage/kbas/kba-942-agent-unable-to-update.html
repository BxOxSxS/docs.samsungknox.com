Agent unable to update

Last updated January 15th, 2024

Categories:

Configuration

Environment

  • Knox Manage
  • Android Enterprise

Overview

You might encounter an issue where an update to the Knox Manage agent on the device is prompted but fails.

Cause

If the minimum requirements for a user’s device are not met and the Update the KM Agent for Android configuration is set to Force update or User consent, the Knox Manage agent update fails.

Resolution

After ensuring that the device meets all of the minimum requirements, you must set Update the KM agent for Android to Manual:

  1. On the Knox Manage console, go to Setting > Configuration > Basic Configuration.
  2. Click Device.
  3. Under Update the KM Agent for Android, select Manual.
  4. Click Save.

You can now update the Knox Manage agent by either accessing the Knox Manage agent on the device or sending the device command Update Knox Manage agent to the device.
