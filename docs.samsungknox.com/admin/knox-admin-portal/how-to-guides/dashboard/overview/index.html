Overview

Last updated June 17th, 2026

The Dashboard is the default landing page that’s displayed when you sign in to the Knox Admin Portal. It’s a centralized hub for quick access to information related to your account, such as licenses, open support tickets, device health and security insights, and so on.

On the dashboard you can view the following:

  • What’s New dialog when you first sign in after a Knox cloud services release. It provides a brief overview of the key updates in the release. You can click Learn more next to a Knox service name to read the full release notes. To revisit this dialog, click > What’s New.
  • Action cards for pending actions that need your attention.
  • Get started section to help you set up your account and add devices. You can re-open it by clicking > Get started.
  • Device alerts tile showing devices with critical issues from each service.
  • Open support tickets tile showing status of any tickets you submitted.
  • Device lifecycle summary tile to track device health and security.
  • Knox highlights tile that shows information about new and important features.
  • Quick links tile where you can add up to six shortcuts to frequently-visited pages.
  • Active licenses tile where you can monitor all your licenses.
  • Other solutions and tools tile for links to download offline apps.

You can access the dashboard at any time by clicking Dashboard in the Knox Admin Portal navigation pane.

You can also customize the dashboard tiles, drag and drop tiles to reorder them, or change the default dashboard to view a service-specific dashboard on sign in. For more information, see Customize the portal.
