Use bulk actions

Last updated June 17th, 2026

The BULK ACTIONS tab provides you with an easier and more efficient way to manage large fleets of devices. To perform any bulk action, you need to first enter device details and any other data in a CSV template in the required format and then upload the file.

Common actions

Assign common tags

You can assign up to 10 common tags to multiple devices in a single bulk action. Use the provided CSV template to add, assign, and remove common tags in bulk.

To download the CSV template, click Assign common tags, and click Download CSV template on the page. You must provide device IMEIs or serial numbers and corresponding common tags in the template. If you specify a new tag, it’s added to the common tag list after upload.

To assign common tags, click Assign common tags, browse and select the CSV file, and click Submit. You can also overwrite or remove any common tags assigned to devices.

Delete devices

You can use a CSV template to delete up to 10,000 devices in a single bulk action.

To download the CSV template, click Delete devices and click Download CSV template on the page.

To delete devices, click Delete devices, browse and select the CSV file, and click Continue. Then provide a reason for the deletion in the Delete devices dialog, and click Delete.

If the deletion is successful, the devices listed in the CSV file are deleted from all supported services in the Knox Admin Portal, including Knox Remote Support. If the deletion fails or is only partially successful, you’ll see a notification with a link to download a CSV file containing more information.

Assign enrollment profiles

You can use a CSV template to assign enrollment profiles and user credentials for up to 10,000 devices in a single bulk action.

To download the CSV template, click Assign enrollment profile and click Download CSV template on the page. For more information about using the CSV template, click How to prepare your CSV file.

To assign enrollment profiles, click Assign enrollment profile, browse and select the CSV file, and click Submit. You can also select Overwrite existing user credentials to replace users’ existing passwords with ones specified in the CSV file.

Unassign enrollment profiles

You can remove enrollment profiles and clear user credentials for up to 10,000 devices using a CSV template.

To download the CSV template, click Unassign enrollment profile and click Download CSV template on the page. You must list the IMEIs or serial numbers of the devices in the CSV file. For more information about using the CSV template, click How to prepare your CSV file on the page.

To remove enrollment profiles, click Unassign enrollment profile, browse and select the CSV file listing the IMEIs or serial numbers of the devices, and click Submit. You can select Unassign user credentials from these devices to remove any user IDs or passwords from the devices.

Reassign license key

You can replace the existing license and reassign a different license key for up to 10,000 devices using a CSV template. Bulk reassignment of licenses is supported for all devices, including the BYOD and non-Samsung devices enrolled in Knox Manage.

Reassigning license keys in bulk is not supported for Knox Configure, Knox Guard, and Samsung Care+ for Business.

To download the CSV template, click Reassign license key and click Download CSV template on the page. You must list the IMEIs or serial numbers of the devices in the CSV file. For more information about using the CSV template, click How to prepare your CSV file on the page.

In case of BYOD devices enrolled with Knox Manage, provide the unique enrollment identifier associated with a device, instead of its IMEI or serial number.

To reassign a license key:

  1. Click Reassign license key in the BULK ACTIONS tab.
  2. Browse and select the CSV file listing the IMEIs or serial numbers of the devices, and click Continue.
  3. Select the license to be reassigned to the devices listed in the CSV file, and click Submit.

Upload EMM credentials

To manage devices enrolled in Knox Mobile Enrollment, Knox Manage, Knox E-FOTA, or Knox Asset Intelligence with an EMM, you must first assign the EMM’s credentials to devices. You can use this feature to set the initial credentials for device users at scale. Ensure that you are assigned the Common > EMM credentials > Create and edit permission.

To assign EMM credentials for up to 100,000 devices using a CSV template:

  1. Click Upload EMM credentials in the BULK ACTIONS tab. The Upload EMM credentials page displays.

  2. Enter the EMM user ID (usually the email address) and corresponding EMM password for each device user in the CSV file.

    • Note that providing an EMM password is optional.
    • You can enter a maximum of 100 characters for the EMM user ID. Ensure that the user ID doesn’t contain these special characters: # / $ * % & : \ ( ) + ? { } [ ] < >.

    For detailed instructions, click How to prepare your CSV file on the page.

  3. Click Browse file and select the CSV file to upload it.

  4. Select Overwrite existing EMM credentials, if required.

  5. Click Upload.

The EMM CREDENTIALS tab in the EMM page opens displaying the uploaded credentials — the EMM passwords are masked for security.

A notification confirms the successful assignment of EMM credentials to selected devices. In case of unsuccessful or partially successful assignments, you can download a CSV file from the notification to view details.

Upload devices

This action is available to you if Knox E-FOTA, Knox Asset Intelligence, and Samsung Care+ for Business are enabled for you in Settings > Show/Hide services. You can upload up to 10,000 devices to these services from the BULK ACTIONS tab.

To upload devices for Knox E-FOTA, Knox Asset Intelligence, and Samsung Care+ for Business, you must have the Devices and Uploads > Manage devices permission for the service.

To upload devices:

  1. Click Upload devices in the BULK ACTIONS tab. The Upload devices page displays.

  2. Click Download CSV template to get the template and enter device details in the specified format.

  3. Click Browse file and select the CSV file to upload.

  4. Click Continue.

  5. (Optional) If uploading devices to Knox E-FOTA, select a campaign and click Continue.

If the device upload is successful, the common device list opens to show the devices uploaded for the specific Knox cloud service. You can download the device list as a CSV file, if required.

All device uploads are displayed on the UPLOADS tab as well.

Knox E-FOTA actions

Assign campaign

Campaigns are used to manage and apply firmware updates on devices enrolled in Knox E-FOTA. If you have the Knox E-FOTA’s Manage devices > Create, assign and unassign campaign permission, you can manage firmware updates on multiple devices by assigning Knox E-FOTA campaigns using a CSV template.

To assign a campaign to multiple devices:

  1. Click Assign campaign in the BULK ACTIONS tab. The Assign campaign page displays.

  2. Click Download CSV template to get the template and enter device IMEIs or serial numbers in the specified format.

  3. Click Browse file and select the CSV file to upload.

  4. Select an existing campaign from the options shown in the table. Click a campaign to view its detailed information.

    Alternatively, click Create campaign to open the Knox E-FOTA console and create a new campaign. For more information, see Create a campaign in the Knox E-FOTA guide.

  5. Click Submit.

The campaign is applied to the selected devices.

Unassign campaign

If you have the Knox E-FOTA’s Manage devices > Create, assign and unassign campaign permission, you can unassign campaigns from multiple devices using a CSV template.

  1. Click Unassign campaign in the BULK ACTIONS tab. The Unassign campaign page displays.
  2. Click Download CSV template to get the template and enter device IMEIs or serial numbers in the specified format.
  3. Click Browse file and select the CSV file to upload.
  4. Click Unassign.

The campaigns are removed from the devices. There is no change to the license in use on a device, however the device firmware is locked to its current version.

Knox Asset Intelligence actions

Device grouping

You can use the Device grouping feature to efficiently organize and manage the devices enrolled in Knox Asset Intelligence. By categorizing devices into logical groups, you can streamline operations, apply bulk actions, and enhance visibility across your device fleet.

If you have Knox Asset Intelligence’s Manage devices > Manage groups permission, you can assign groups to multiple devices using a CSV template:

  1. Click Device grouping in the BULK ACTIONS tab. The Device grouping page displays.
  2. Click Download CSV template to get the template and enter device and group details in the specified format. You can enter up to 10,000 devices in the CSV.
  3. Click Browse file and select the CSV file to upload.
  4. Click Submit.

The updated device groups information is displayed in the common device list.

Knox Configure actions

Assign profile

With Knox Configure profiles, you can configure settings, such as restrictions, apps, and content, for your enterprise devices when they’re deployed or updated. For more information, see About Knox Configure profiles.

You can assign profiles to devices enrolled in Knox Configure if:

  • You have Knox Configure’s Manage devices > Assign and push update permission.
  • You have Knox Configure’s Manage Licenses > Add and edit permission to assign licenses.
  • The devices are assigned a dynamic profile.
  • The devices are in the Unassigned, Profile assigned, Error, Cancelled by user, Configured, Failed to configure, or Updates pushed status.

To assign profiles to multiple devices using a CSV template:

  1. Click Assign profile in the BULK ACTIONS tab. The Assign profile page displays.
  2. Click Download CSV template to get the template and enter device IMEIs or serial numbers in the specified format. You can enter up to 10,000 devices in the CSV.
  3. Click Browse file and select the CSV file to upload.
  4. Select a profile to assign to the devices.
  5. Select Push update without requesting end user consent to apply the profile automatically. This option is available only if the selected profile has advanced settings configured.
  6. Click Next. The Select license page displays.
  7. Select a license to assign to the devices. You must select a license with enough remaining license seats for your devices.
  8. (Optional) If the license does not have enough seats for the number of devices you selected, use the Enter license key or Get a license options on the Select license page to get additional licenses. For more information, see Enter a license key and Get a license in the Knox Configure guide.
  9. Click Assign.

The profile is applied to the devices listed in the CSV file.

Unassign profile

You can unassign a profile to remove all device configurations from devices enrolled in Knox Configure if:

  • You have Knox Configure’s Manage devices > Assign and push update permission.
  • The devices are not locked.

To unassign profiles from devices using a CSV template:

  1. Click Unassign profile in the BULK ACTIONS tab. The Unassign profile page displays.
  2. Click Download CSV template to get the template and enter device IMEIs or serial numbers in the specified format. You can enter up to 10,000 devices in the CSV.
  3. Click Browse file and select the CSV file to upload.
  4. Click Submit.

A notification confirms the removal of profiles from devices, and the devices status changes to Unassigned. In case of unsuccessful or partially successful assignments, you can download a CSV file from the notification to view details.

The license seats that are disassociated from devices are returned to the license pool and become available for assignment to other devices.

Push profile update

You can push profile updates to devices enrolled in Knox Configure if they meet the following conditions:

  • You have Knox Configure’s Manage devices > Assign and push update permission.
  • The devices are assigned a dynamic profile.
  • The devices are in the Configured, Updates pushed, Failed to configure, or Failed state.

To push profile updates to multiple devices using a CSV template:

  1. Click Push profile update in the BULK ACTIONS tab. The Push profile update page displays.
  2. Click Download CSV template to get the template and enter device IMEIs or serial numbers in the specified format.
  3. Click Browse file and select the CSV file to upload.
  4. Specify the updates to deploy:
    • Only changed items
    • Full profile update
  5. Configure notifications and consent management:
    • Request consent from device user
    • Request consent and show custom notification. This option is available only if all selected devices have the same Dynamic profile and use Knox 3.4 or higher in Normal or ProKiosk mode. You can set the following additional options:
      • Select existing notification: Select from the options available.
      • Create new notification: The notification version is auto-generated. Specify the text for the new notification. Optionally, you can also specify a hyperlink to include in the notification.
    • Push update without requesting end-user consent
  6. Select from the following options to configure when to set the profile updates:
    • Push now
    • Schedule date and time: Specify the Date, Time, and Time zone for the profile update.
  7. Click Push update.

Lock devices

If you have Knox Configure’s Manage devices > Lock permission, you can lock devices to prevent misuse. Only the devices in the Configured, Updates pushed, Failed, and Failed to configure states can be locked.

To lock devices enrolled in Knox Configure using a CSV template:

  1. Click Lock devices in the BULK ACTIONS tab. The Lock devices page displays.
  2. Click Download CSV template to get the template and enter device IMEIs or serial numbers in the specified format.
  3. Click Browse file and select the CSV file to upload.
  4. Click Next.
  5. Select one of the following options to specify the contact details displayed to device users to help unlock their devices:
    • Default: Contact information associated with assigned profile
    • Add custom contact information. Specify company name, phone number, and email in the additional fields that display.
  6. Click Lock devices.

The status of the devices is changed to Locked in the common device list.

Unlock devices

If you have Knox Configure’s Manage devices > Unlock permission, you can unlock devices in the Locked status.

To unlock multiple devices enrolled in Knox Configure using a CSV template:

  1. Click Unlock devices in the BULK ACTIONS tab. The Unlock devices page displays.
  2. Click Download CSV template to get the template and enter device IMEIs or serial numbers in the specified format.
  3. Click Browse file and select the CSV file to upload.
  4. Click Unlock devices.

The devices are unlocked and their status is changed to Unlocked on the common device list.

Reboot or factory reset devices

You can reboot or factory reset devices enrolled in Knox Configure if they are in the Configured, Updates pushed, or Failed to configure state.

To reboot or factory reset multiple devices using a CSV template:

  1. Click Reboot or factory reset devices in the BULK ACTIONS tab. The Reboot or factory reset devices page displays.
  2. Click Download CSV template to get the template and enter up to 10,000 device IMEIs or serial numbers in the specified format.
  3. Click Browse file and select the CSV file to upload.
  4. Specify the action to perform on the devices:
    • Reboot devices to reboot devices without device user consent
    • Factory reset devices to restore the devices to their original out-of-the-box state
    • Factory reset and eSIM deletion to restore devices to their original out-of-the-box state and also remove any existing eSIM configurations
  5. Click Submit.

A notification displays to confirm the reboot or factory reset. In case of unsuccessful or partially successful assignments, you can download a CSV file from the notification to view details.

Assign custom ID

If you have Knox Configure’s Manage devices > Manage Custom ID permission, you can assign custom IDs to devices enrolled in Knox Configure for easier identification.

To assign custom IDs to multiple devices using a CSV template:

  1. Click Assign custom IDs in the BULK ACTIONS tab. The Assign custom IDs page displays.
  2. Click Download CSV template to get the template and enter the devices’ IMEIs or serial numbers and respective custom IDs for up to 10,000 devices in separate columns.
    • The maximum supported length of Custom IDs is 50 characters.
    • Ensure that the custom ID doesn’t contain spaces or special characters, including # / $ * % ^ & : \ ( ) + ? { } [ ] < >.
  3. Click Browse file and select the CSV file to upload.
  4. Click Assign custom IDs.

The custom IDs for each device are displayed in the common device list.

Samsung Care+ for Business actions

Assign license key

You can assign a Samsung Care+ for Business license to activate up to 10,000 devices using a CSV template.

To download the CSV template, click Assign license key and click Download CSV template on the page. You must list the IMEIs or serial numbers of the devices to be activated in the CSV file. For more information about using the CSV template, click How to prepare your CSV file on the page.

To assign a Samsung Care+ for Business license:

  1. Click Assign license key in the BULK ACTIONS tab.
  2. Browse and select the CSV file listing the IMEIs or serial numbers of the devices.
  3. Select a Samsung Care+ for Business license, and click Submit.
