Manage EMM credentials

Last updated June 17th, 2026

To manage devices enrolled in Knox Mobile Enrollment, Knox Manage, Knox E-FOTA, or Knox Asset Intelligence with an EMM, you must first assign the EMM’s credentials to devices.

The EMM CREDENTIALS tab of the ENROLLMENT page displays a table of users that can be assigned to devices. You can add users, edit or delete passwords, and import user IDs and passwords using a CSV file.

The EMM credentials table displays information such as:

  • EMM user ID: The unique name or identifier of the user.
  • EMM password: The user’s password.

You can view this information when you select a device in the common or Knox Mobile Enrollment device list.

Add EMM credentials

To add EMM credentials for a device:

  1. Click Enrollment on the Knox Admin Portal navigation pane. The Enrollment page displays.

  2. In the EMM CREDENTIALS tab, click Add EMM credentials. The Add EMM credentials dialog displays.

  3. Enter the EMM user ID for the device user.

  4. (Optional) Enter the corresponding EMM password for the device user.

  5. Click Add EMM credentials.

The EMM CREDENTIALS tab displays the added credentials.

Edit EMM credentials

To edit the EMM password for a device user:

  1. Click Enrollment on the Knox Admin Portal navigation pane. The Enrollment page displays.

  2. On the EMM CREDENTIALS tab, click the device user’s EMM user ID. The Edit EMM credentials dialog displays.

  3. Enter the password for the device user. The password must be the same as the credentials defined by your EMM.

  4. (Optional) Click Check password. The Check password dialog displays. You can use this dialog to verify the device user’s password if there’s a password conflict during the sign-in process.

  5. Enter a password to compare with the password recorded in the Knox Admin Portal and click Compare.

    If the password matches, a Passwords match message displays. If they don’t match, Passwords do not match displays. You can click Update to change the current password to one you’ve entered for comparison.

  6. Click Update to save the password you entered.

The EMM CREDENTIALS tab displays the added credentials.

Delete EMM credentials

To permanently delete one or more EMM credentials:

  1. On the EMM CREDENTIALS tab, select the EMM credentials to be deleted.

  2. Click Actions > Delete EMM credentials. The Delete EMM credentials dialog displays.

  3. Click Delete.

Download EMM credentials as CSV

To download EMM credentials as a CSV:

  1. On the EMM CREDENTIALS tab, select the EMM credentials.

  2. Click Actions > Download EMM credentials as CSV.
