Knox Admin Portal 23.12 release notes

Last updated November 23rd, 2023

This document is new for the Knox cloud services 23.12 UAT.

On this tab

  • Unified email preferences
  • Improvements to license support for MSP admins
  • Link to Knox Remote Support device list
  • Security update for PII storage and search

Unified email preferences

Previously, it was only possible to set email preferences for Knox Mobile Enrollment, Knox E-FOTA, and Knox Guard on separate pages in the Knox Admin Portal settings.

With Knox Admin Portal 23.12, you can now set which types of emails you want to receive from individual Knox services from Email preferences. You can also choose to receive emails about information common to the Knox Admin Portal, such as account and admin details.

If you previously specified email preferences for Knox Mobile Enrollment or Knox E-FOTA, those settings are preserved in the unified email preferences.

See Configure your account settings to learn more.

Improvements to license support for MSP admins

With Knox Admin Portal 23.12, MSP admins can now manage other Knox licenses on the Knox Admin Portal, such as Knox Platform for Enterprise and Knox Capture licenses.

On behalf of their managed customers, MSP admins can:

  • Generate Knox Platform for Enterprise Premium license keys
  • Renew Knox Platform for Enterprise Premium license keys
  • Generate Knox Capture Scandit Edition trial license keys

MSPs and managed customers can see these licenses on the managed customer’s Knox Admin Portal, but they won’t be shown on the Knox MSP Portal.

Link to Knox Remote Support device list

If a device previously connected to Knox Remote Support at least once, its unified device info page now shows Knox Remote Support in the list of services.

From the unified info page, you can click the service name, or click Activated in the DEVICE STATUS column for Knox Remote Support, to open a new page that contains a filtered view of the Knox Remote Support device list.

Alternatively, from the Licenses page, you can click the tooltip in the ACTIVATED column for a Knox Suite license, then click the Knox Remote Support activation count to see the list of devices activated with a Knox Suite license key.

See Manage Knox Remote Support devices in the Knox Remote Support admin guide for more details.

Security update for PII storage and search

This release delivers a significant update to Knox cloud infrastructure that further hardens how fields with personally identifiable information (PII) are stored. This update impacts how you can search for PII across all Knox services.

For the technical details about this update, as well as an up-to-date breakdown of how this affects Knox Admin Portal, see Security notice regarding storage and search behavior for PII.
