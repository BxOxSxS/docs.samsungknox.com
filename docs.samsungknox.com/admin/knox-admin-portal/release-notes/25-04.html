Knox Admin Portal 25.04 release notes

Last updated April 9th, 2025

New

Manage cross-service actions from Devices page

Managing devices across Knox cloud services is now easier on the Knox Admin Portal. You can now initiate the following cross-service actions directly from the common Devices page, without needing to navigate to each service.

  • Assign or unassign Knox Mobile Enrollment profiles
  • Assign or unassign Knox E-FOTA campaigns
  • Assign or unassign Knox Configure profiles

Delete devices across all Knox cloud services

Effortlessly delete devices across Knox cloud services from the Knox Admin Portal. Use the Devices page to delete devices from Knox Mobile Enrollment, Knox Manage, Knox E-FOTA, Knox Asset Intelligence, Knox Configure, Knox Remote Service, and Samsung Care+ for Business all at once, without having to navigate to each individual console.

You can manually select and delete up to 100 devices, or use a CSV to delete up to 10,000 devices in bulk. If you’re a Managed services provider (MSP), you can delete devices only from the services you manage. You can also check the deletion status updated as Full Delete, Partial Delete, or Failed Delete. For more information, see View and manage devices.

Reassign license keys for up to 100 devices

Reassign license keys for up to 100 devices, using the Reassign license key option on the Devices page. You can view and assign only the licenses that you have permission for and that have enough license seats for the selected devices.

This feature is supported for Knox Suite Plans and for Knox E-FOTA, Knox Asset Intelligence, Knox Manage (for wearables only) license keys. For more information, see View and Manage Devices.

Common roles for admins

For streamlined and efficient workflows, create and assign common roles to admins with permissions to manage common tags and delete devices from the common device list. To do so, select Common tags and Delete devices when setting the Manage devices permission for a role. For more information, see Manage admins and roles for Knox services.

Support for Portuguese

In addition to currently-supported languages, you can also view the Knox Amin Portal in Portuguese. If set, any Knox service emails are also sent in Portuguese.

User session extension and custom timeout duration

You can now view the Session expires in timer in the footer of the Knox Admin Portal and refresh it to easily extend your session — the default timeout duration is 60 minutes. Super-admins can change the timeout duration for all users in their tenant on the Settings page.

Updates

View cross-service notifications in new common category

Use the new Common section in the notification center to review common, cross-service events, such as those related to the Delete devices, Reassign license key, Assign common tags, and Remove common tags actions. You only receive one notification for events that apply across all Knox cloud services.
