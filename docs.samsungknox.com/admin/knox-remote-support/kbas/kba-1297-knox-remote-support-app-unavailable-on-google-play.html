Knox Remote Support app unavailable on Google Play

Last updated September 23rd, 2024

Environment

  • Knox Remote Support
  • Google Play

Overview

When adding the Knox Remote Support app to Managed Google Play, you might encounter an issue where the Knox Remote Support app is unavailable for download.

Cause

In preparation for the new release, the Knox Remote Support app was removed from Google Play on September 22nd, 2024.

The Knox Remote Support app is set to be republished starting September 25th, 2024 as part of the Knox cloud services 24.09 release.

Workaround

If you need to use the previous version of the Knox Remote Support app, you can install it using the following methods.

You can download a standalone APK for the Knox Remote Support app to install it on your device. If you’re using Knox Manage, you can add the APK to your Knox Manage console as an internal app:

Region URL
Asia https://install-ap.manage.samsungknox.com/KnoxManageRemoteSupportClient.apk
Europe https://install-eu.manage.samsungknox.com/KnoxManageRemoteSupportClient.apk
Americas https://install-us.manage.samsungknox.com/KnoxManageRemoteSupportClient.apk

If using Knox Manage, you can also install the Knox Remote Support app using the Knox Manage agent on the device.

  1. On the device, launch the Knox Manage agent.
  2. On the top left of the screen, expand the menu.
  3. Tap Service Desk.
  4. Tap Download Knox Remote Support App to download the Knox Remote Support app to your device.
  5. Once the Knox Remote Support app is installed, tap Run Knox Remote Support App to run the app.
