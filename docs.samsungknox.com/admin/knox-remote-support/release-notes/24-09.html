Knox Remote Support 24.09 release notes

Last updated September 12th, 2024

This document was updated for the Knox cloud services 24.09 UAT.

On this tab

  • New
  • Support for Role-Based Access Control for MSP admins
  • Support for Knox Remote Support agent in work profile
  • Updates
  • Update to license assignment logic
  • Consistent account information

New

Support for Role-Based Access Control for MSP admins

With Knox Remote Support 24.09, MSP admins can now use Role-Based Access Control (RBAC) to control the admin permissions on behalf of their customers. They can define and assign specific roles to Knox Remote Support admins and grant them access to specific features, as required. The Create role action in the Knox MSP Portal lets you create new roles and grant them access to features such as Devices list, Remote support session initiation, Remote support device control, and Administrators and Roles.

For more information, see Manage roles.

Support for Knox Remote Support agent in work profile

Previously, the Knox Remote Support agent installation was limited to the device’s personal profile. Starting with Knox Remote Support 24.09, the agent can now also be installed in a device’s work profile. This feature enables the device users to utilize selective Knox Remote Support functions within the work profile itself, ensuring that access to the personal profile data remains secure and separate.

Updates

Update to license assignment logic

Starting with Knox Remote Support 24.09, if a device is already enrolled in services that are a part of Knox Suite, then the device is enrolled in Knox Remote Support with the existing Knox Suite license key. Previously, Knox Manage license key was preferred for enrollment.

Consistent account information

Knox Remote Support 24.09 onwards, if you’ve a Knox Admin Portal account, then regardless of you accessing Knox Remote Support from the Knox Manage console or Knox Admin Portal, the account information displayed remains consistent. When you click on your account name in Knox Remote Support, it shows Samsung Account, Customer ID, and Secret Key information.
