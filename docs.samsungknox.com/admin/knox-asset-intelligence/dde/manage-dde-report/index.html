Manage Dynamic Data Export report

Last updated November 19th, 2025

The Dynamic Data Export page allows you to view and manage your custom reports. Once a report is created, it appears in the table on the page.

The table consists of following information:

  • REPORT NAME — The name that you gave to the report. Note that multiple reports can exist with the same name. Click the link to open the report details panel.
  • DESCRIPTION — The description you provided for the report.
  • STATUS — The status of the report:
    • COMPLETED — The system successfully created the report. This status always includes a report download link where you can download the report as a CSV file.
    • IN PROGRESS — The system is creating the report.
    • FAILED — The system failed to generate the report due to one of the following reasons:
      • The system created the report, but there was no data available for the selected criteria.
      • The system failed to generate the report.
      • The selected criteria resulted in a report that exceeded the maximum data size (more than 1 million rows).
      • The report creation timed out after 6 hours.
      • The report failed to generate unknown reasons, please try again.
    • EXPIRED — The system was able to create the report, but the report download link is older than 30 days and has expired.
  • CHART TYPE — The type of chart selected for the report: Table, Line, or Pie.
  • DATE RANGE — The selected date range when defining the report criteria.
  • CREATED BY — The email address of the admin who created the report.
  • CREATED DATE — The date and local time the report was created.

Report details panel

You can click any report name to see its details in an expanded side panel, where you can review the same information from the table, as well as additional report criteria like its measures, calculations, dimensions, and interval. Additionally, if you are reviewing a Line or Pie type report, you’ll get an interactive preview of the chart where you can show and hide reporting measures and view data points.

From the details panel, you can take the following actions. Note that the same actions can be quickly access from the ACTIONS dropdown button.

  • Click EDIT to open the Dynamic Data Export report creator where you can make edits to your existing report. Once you make your changes, click SAVE to return to the table.

  • Click the Options button ( ) to perform the following additional actions:

    • Duplicate the report. This option will open the Dynamic Data Export report creator, with all fields pre-populated. Make optional changes if required, then click CREATE to generate the report and return to the table.
    • Download as CSV. This option provides the same CSV file as the download link when viewing a report with the Completed status.
    • Download as PNG. This option is only available for Line and Pie type reports, and allows you to download the chart as a PNG image.
    • Delete. This deletes the report from the system.
