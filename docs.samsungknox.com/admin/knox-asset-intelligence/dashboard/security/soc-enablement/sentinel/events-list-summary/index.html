Security events

Last updated July 14th, 2025

The following table describes the available Essential and Advanced security events that Knox Asset Intelligence can collect and report to Microsoft Sentinel.

Essential security events

Event Name Event Description
TAG_FAILED_TO_WIPE_USER_DATA Indicates the process of wiping user data on the device failed for a specific reason
TAG_WIPING_DATA_IS_NOT_ALLOWED_FOR_THIS_USER Indicates the process of wiping data (factory reset) is not allowed for this user
TAG_ADMIN_HAS_REQUESTED_FULL_WIPE_OF_DEVICE Indicates an administrator requested full wipe of device
LOG_IS_FULL Indicates the on-device Knox Security Log is full
TAG_ADB_SHELL_INTERACTIVE Indicates an ADB interactive shell was opened via “adb shell”
TAG_KEYGUARD_DISABLED_FEATURES_SET Indicates an administrator has disabled keyguard features set
TAG_KEYGUARD_DISMISSED Indicates the keyguard has been dismissed
PREVENT_APP_REMOVAL_CAPABILITY Indicates when an app removal is prevented
KEY_INPUT_CAPTURE_CAPABILITY Indicates when the key input capture permission for an app is enabled
USER_INTERACTION_CONTROL_CAPABILITY Indicates when the user screen control permission in an app is enabled
BOOT_COMPROMISED_SOFTWARE_BINARY Indicates the device boot binary is at risk of compromise
BOOT_STATE Indicates the device boot state
BOOT_SECURITY_ABNORMALITY Indicates the device boot is found to have a security abnormality
PERIPHERAL_ACCESS_THROUGH_POLICY_DETECTED_CAMERA Indicates when the device camera access has been detected while it is disabled by a system policy
PERIPHERAL_ACCESS_THROUGH_POLICY_DETECTED_MIC Indicates when the device microphone access has been detected while it is disabled by a system policy
PASSWORD_LOCKOUT Indicates when the device is locked out after the user has reached maximum password attempts

Advanced security events

Event Name Event Description
TAG_PACKAGE_NAME_HAS_BEEN_ACTIVATED_AS_ADMIN Indicates the application was activated as administrator
TAG_APPLICATION_ACTION_FAILED_BECAUSE_OF_SIGNATURE_VERIFICATION_FAILURE Indicates the application action has failed because of signature verification failure
TAG_PACKAGE_NAME_HAS_BEEN_REMOVED_AS_ADMIN Indicates the application was removed as administrator
SCREEN_CAPTURE_CAPABILITY Indicates when the use of device screen capture permission for an app is enabled
ACCESS_NOTIFICATION_PERMISSION Indicates when permission to access/manage notification in an app is enabled
ACCESS_CALL_LOG_PERMISSION Indicates when an app has permission to access call logs on launch
PROCESS_PRIVILEGE_ESCALATION Indicates when an app has transitioned from an acceptable uid/esuid/fsuid to a non-app id
TAG_NETWORK_EVENT_INSECURE_PACKET Indicates apps that are using any of the following insecure protocols: HTTP(80), FTP(20/21), TELNET (23), SMTP(25), POP3(110), IMAP3(143), RADIUS(1812/1813/1645,1646)
SUSPICIOUS_URL_ACCESSED Indicates when the user has tapped or clicked on a potentially suspicious URL on the device
TAG_ADMIN_HAS_ALLOWED_TO_INSTALL_APPLICATION Indicates an administrator allowed application install
TAG_ADMIN_HAS_CHANGED_NFC_STATE_CHANGE Indicates an administrator has allowed the NFC state change
TAG_ADMIN_HAS_ENABLED_BLUETOOTH_DISCOVERABLE_STATE Indicates an administrator enabled bluetooth discoverable state
TAG_ADMIN_HAS_ENABLED_WIFI_DIRECT Indicates an administrator enabled wifi direct
TAG_ADMIN_HAS_ALLOWED_CAMERA Indicates an administrator allowed camera
TAG_MICROPHONE_ENABLED Indicates the microphone is enabled
TAG_ADMIN_HAS_ALLOWED_MICROPHONE Indicates an administrator allowed microphone
TAG_ADMIN_HAS_DISALLOWED_MICROPHONE Indicates an administrator disallowed microphone
TAG_ADD_UNTRUSTED Indicates an administrator added a certificate to the trusted database
TAG_ADMIN_HAS_CHANGED_LOCK_SCREEN_STATE_TO_DISABLED Indicates an administrator changed lock screen state to disabled
TAG_ADMIN_HAS_CHANGED_SCREEN_LOCK_TIME_OUT Indicates an administrator changed screen lock time out
TAG_ADB_SHELL_CMD Indicates that a shell command was issued over ADB via adb shell
TAG_APP_PROCESS_START Indicates an application process was started
TAG_BACKUP_SERVICE_TOGGLED Indicates an administrator has enabled or disabled backup service
TAG_BLUETOOTH_CONNECTION Indicates the device attempts to connect to a Bluetooth device
TAG_CERT_AUTHORITY_INSTALLED Indicates a new root certificate has been installed into system’s trusted credential storage
TAG_CERT_AUTHORITY_REMOVED Indicates a new root certificate has been removed from system’s trusted credential storage
TAG_KEYGUARD_DISMISS_AUTH_ATTEMPT Indicates there has been an authentication attempt to dismiss the keyguard
TAG_MEDIA_MOUNT Indicates removable media has been mounted on the device
TAG_PACKAGE_INSTALLED Indicates a package is installed
TAG_PACKAGE_UNINSTALLED Indicates a package is uninstalled
TAG_PACKAGE_UPDATED Indicates a package is updated
TAG_REMOTE_LOCK Indicates an administrator remotely locked the device or profile
TAG_SYNC_RECV_FILE Indicates a file was pulled from the device via the adb daemon, for example via adb pull
TAG_SYNC_SEND_FILE Indicates a file was pushed to the device via the adb daemon, for example via adb push
TAG_WIPE_FAILURE Indicates a failure to wipe device or user data
ACCESSIBILITY_EVENT Indicates the accessibility api is called for an app
RESTRICTED_PERMISSION Indicates the launched app has ‘restricted permission’
TAG_APPLICATION_INSTALLATION_NOT_ALLOWED_BY_ADMIN_BLOCKLIST Indicates the application is being blocked from installation by a device policy enforced by an administrator
TAG_APPLICATION_INSTALLATION_NOT_ALLOWED_BY_ADMIN_INSTALLER_BLOCKLIST Indicates that an administrator has blocked the installation of an application from a specific installer
TAG_BIND_TO_VPN_FAILED_COULD_NOT_FIND_PACKAGE Indicates when a bind to VPN vendor service failed as vendor package could not be found
TAG_ERROR_OCCURRED_WHILE_VALIDATING_PROFILE_INFORMATION_FOR_VENDOR Indicates that during VPN profile creation, an error occurred while validating vendor
TAG_ADMIN_HAS_REMOVED_SSID_FROM_THE_RESTRICTION_BLOCKLIST Indicates an administrator removed a SSID from restriction blocklist
TAG_ADMIN_HAS_REMOVED_ALL_SSID_FROM_THE_RESTRICTION_BLOCKLIST Indicates an administrator removed all SSIDs from restriction blocklist
TAG_ADMIN_HAS_ADDED_SSID_TO_THE_RESTRICTION_ALLOWLIST Indicates an administrator added a SSID to restriction allowlist
TAG_ADMIN_HAS_ADDED_TO_CAMERA_ALLOWLIST Indicates an administrator added package and signature to camera allowlist
TAG_REMOVE_UNTRUSTED Indicates an administrator removed a certificate from the untrusted database
TAG_APPLICATION_INSTALLATION_NOT_ALLOWED_BECAUSE_SIGNED_UNTRUSTED_CA Indicates an app installation is not allowed because it is signed by an untrusted CA
TAG_ADMIN_HAS_LOCKED_WORKSPACE Indicates an administrator locked workspace
TAG_ADMIN_HAS_UNLOCKED_WORKSPACE Indicates an administrator unlocked workspace
TAG_ADMIN_HAS_SUCCESSFULLY_LOCKED_WORKSPACE Indicates an administrator successfully locked workspace
TAG_ADMIN_HAS_SUCCESSFULLY_UNLOCKED_WORKSPACE Indicates an administrator successfully unlocked workspace
TAG_KEY_INTEGRITY_VIOLATION Indicates a failed cryptographic key integrity check
TAG_LOG_BUFFER_SIZE_CRITICAL Indicates that the audit log buffer has reached 90% of its capacity
TAG_MEDIA_UNMOUNT Indicates that removable media was unmounted from the device
TAG_PASSWORD_CHANGED Indicates the user has just changed their lockscreen password
TAG_PASSWORD_COMPLEXITY_REQUIRED Indicates an administrator has set a password complexity requirement, using the platform’s pre-defined complexity levels
TAG_PASSWORD_COMPLEXITY_SET Indicates an administrator has set a requirement for password complexity
TAG_USER_RESTRICTION_REMOVED Indicates an administrator has removed a user restriction
SUSPICIOUS_URL_DETECTED Indicates when the user has copied a potentially suspicious URL on the device
