Total, Background, and Foreground app usage

Last updated December 19th, 2024

The Total app usage, Background app usage, and Foreground app usage insights lets you know which apps in your fleet have the most usage over a selected reporting period.

These insights do NOT show data when the reporting period is set to Today.

Dashboard tile

The dashboard tiles for the Total, Background, and Foreground app usage each display the top five most-used apps in their respective categories.

A bar chart displays each of the top five apps, with each segment corresponding to the list below. Hover over each segment of the bar chart to view the total, background, and foreground usage hours for a specific app, as well as the number of devices using the app.

Click the expanded view button (>) to see total app data for all apps in your fleet. To drill down into usage details for any of the top five apps, click an app name in the list.

Expanded details view

The expanded view pages for Total app usage, Background app usage, and Foreground app usage each display data for every app used in your fleet, over a selected reporting period.

Chart

The expanded view chart doesn’t support the Today and Yesterday reporting periods.

On the chart, you’ll see a graph of the top five apps with the most usage hours over a select reporting period. Hover over a chart segment to view a breakdown of the data, separated into total, background, and foreground hours. Click any app in the chart legend to show or hide the data on the graph.

Application list

The application list table doesn’t support the Today reporting period.

In the APPLICATION LIST table, you’ll see usage data for every app used during the selected reporting period. By default, the APP and TOTAL/BACKGROUND/FOREGROUND RUN TIME columns are always displayed. Click the CUSTOMIZE TABLE button (…) to add, remove, or change the columns on the table.

To help you analyze your data offline, you can click the DOWNLOAD AS CSV button to get a copy of the table sent to your email, or click the CREATE APP USAGE REPORT button to build a custom report based on your data requirements.

Create app usage reports

To help you further analyze data related to how apps are used in your fleet, you can generate an app usage report for a specific insight. To do this:

  1. Go to the expanded view page for that insight, then in the APPLICATION LIST table, select up to 10 apps to view in the report.
  2. Click CREATE APP USAGE REPORT.
  3. In the pop-up window, select the reporting period, either Yesterday, or Last 7 days.
  4. In the Data Type field, select the type of data you want to include in your usage report, then click DOWNLOAD USAGE REPORT. The report is sent to your email once ready.

Alternatively, you can generate an app usage report from the Devices page. To do this:

  1. Go to the Devices page, then click a DEVICE ID to open the expanded device details panel.
  2. In the APPS field, click CREATE APP USAGE REPORT, then follow the same procedure above to create your report.

Drill-down page

The Total app usage, Background app usage, and Foreground app usage insights all share the same drill-down view page.

You can acccess the drill-down view page for these insights by clicking any app in the expanded view table, or by clicking any of the top five apps displayed on the insight’s dashboard tile. Here, you’ll see an app’s total, background, and foreground usage data over a selected reporting period.

In the summary ribbon at the top of the page, you’ll see the total number of devices and device-groups that used this app, as well as usage data including run time, network data usage, and battery consumption.

Chart

This page displays two charts:

  • On the TOTAL RUN TIME AND DEVICE COUNT chart, you’ll see a breakdown of the total foreground and background usage per day in hours, along with the corresponding device count. Hover over a chart segment to view additional details and dowload the device list for that day. Click a data type in the chart legend to show or hide the data on the graph.

  • On the DEVICE COUNT PER APP VERSION chart, you’ll see a breakdown of the app versions deployed in your fleet. Hover ovr each segment of the chart to view the total number of devices running the particular app version, as well as run time, battery, and network usage data.

Table

In the drill-down view table, you’ll see a list of every app version deployed in your fleet. By default, the APP VERSION and TOTAL,BACKGROUND,FOREGROUND RUN TIME columns are always displayed (depending on the selected insight). Click the CUSTOMIZE TABLE button (…) to add, remove, or change the columns in this table.

To help you analyze your data offline, you can click the DOWNLOAD AS CSV button to get a copy of the table sent to your email.
