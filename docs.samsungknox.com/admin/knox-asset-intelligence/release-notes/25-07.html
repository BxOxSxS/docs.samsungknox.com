Knox Asset Intelligence 25.07 release notes

Last updated August 25th, 2025

New

Support for device health score (Labs)

A new Labs feature in the device details panel lets you view a health score that lets you know if your device is performing optimally, or if it has issues that require corrective actions. See Device health to learn more about this new Labs feature.

General availability of Battery auto optimization

Battery auto-optimization during workshift, previously a Labs feature, is now release for general availability to all customers. This feature lets you remotely configure the settings of your enterprise devices in order to help extend battery life and reduce downtime associated with having to charge devices during a workshift. See the Battery settings page to learn more.

General availability of Microsoft Sentinel integration

The Microsoft Sentinel Integration feature is now generally available to all customers. This feature provides enterprise IT admins and SecOps (security operations) teams with the ability to capture security telemetry data from Samsung Galaxy devices, and have that data converted into events and logs in Microsoft Sentinel. See the Micosoft Sentinel Overview page to learn more.

Updates

Unified device enrollment across multiple Knox services

To provide a better experience for customers of multiple Knox cloud services, several changes were made to the way Knox Asset Intelligence handles device enrollment. Now, you can enroll devices into multiple services at the same time using the Enrollment profiles. See the Manage Enrollment Profiles page to learn more.

Unified Resellers page

Previously, if you used multiple Knox services, you had to approve reseller uploads and set reseller preferences for each service independently. Now, customers of multiple Knox cloud services can approve all device uploads, and configure reseller preferences from a single, unified Resellers page. See the Manage Resellers page to learn more.

Additional CSV columns for Battery levels at shift start/end insights

When downloading a CSV version of the Battery levels at shift start/end insight, you’ll now see a Battery remaining time column letting you know the remaining hours and minutes before the device loses power. See the battery insight page to learn more.

Microsoft Sentinel connection status updates

The Security Center now continuously monitors the connection status of your Microsoft Sentinel integration. If a connection to Microsoft Sentinel fails, an error message indicating the cause of failure displays on the Settings page. See Security settings to learn more.

Microsoft Sentinel certificate expiration reminders

To help prevent a disconnection with your Microsoft Sentinel environment, you’ll now receive a reminder email when your authentication certificate nears its expiration date. By default, the reminder is sent 60 days before certificate expiration. See Security settings to learn more.
