Knox Asset Intelligence 24.09 release notes

Last updated September 25th, 2024

New

Support for insight on battery charging issues

Knox Asset Intelligence 24.09 provides a new Charging issues dashboard insight that lets you know if battery charging is being affected by factors related to charging accessories, connection issues, or environmental conditions like water and dust ingress. You can use this information to take corrective actions to help mitigate charging issues and increase the overall battery health of your devices. To learn more about this new feature, see Charging issues.

Ability to collect CPU usage for apps and devices (Labs)

The Knox Asset Intelligence 24.09 release adds two new Lab experiment features that let Knox Asset Intelligence collect CPU usage data from your fleet’s apps and devices respectively. This data is used to provide valuable CPU usage dashboard insights in a future release. You can choose to enable the collection of CPU usage data by going to dashboard Settings > DATA CONDITIONS > DATA UPLOAD. See Configure dashboard settings for more details.

Support for insight on Wi-Fi calling status

If your device users connect to a Wi-Fi network to make phone calls, you can now use the Wi-Fi calling setting status dashboard insight to view which devices have the Wi-Fi calling feature enabled or disabled, or which devices are incompatible with the feature. To learn more about this new feature, see Wi-Fi calling setting status.

Updates

Improvements for unified activity logs

Previously, you had to review activity logs for each service on separate pages. Starting with the Knox Asset Intelligence 24.09 release, the Activity log page is now unified for all Knox cloud services, letting you review and download activity data for all services from a single page. To learn more about this new unified activity log, see View activity logs.

Enhancements to Knox Asset Intelligence agent app

Knox Asset Intelligence 24.09 includes an updated agent application, providing an improved user interface that makes it even easier to view the device’s network, app, or battery related diagnostics, and send error reports and logs back to IT admins for troubleshooting. See Use the agent app to learn more.

Improvements to agent app event descriptions

With Knox Asset Intelligence 24.09, the event descriptions on the agent app’s Diagnostics > Apps > Events > Abnormal screen were improved to provide clearer and easier-to-understand information to help users and IT admins troubleshoot issues more effectively.

Support for Security center vulnerability-to-device mapping

The Security center now lets you identify which vulnerabilities are internally verified, patched, and mapped to known device models, and which vulnerabilities are disclosed but awaiting internal confirmation and security patch release. See Security center to learn more about this update.

Deprecations

Sunsetting of Scanning performance and Indoor location tracking features

As part of our product development process, we regularly assess research and development efforts to ensure that we continually align wth our customers’ needs.

Due to limited usage and technical limitations, we removed the Scanning performance dashboard insight, as well as the Location dashboard’s Indoor location tracking features (which include the ability to create and manage venues, view venue analytics, and locate devices inside venues) from the Knox Asset Intelligence 24.09 release.

As a result, these features — as well as their related pages and dashboard settings — are no longer available. There is no disruption to the rest of your Knox Asset Intelligence service as a result of these changes.
