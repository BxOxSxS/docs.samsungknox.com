Knox Capture v2.0 release notes

Last updated July 10th, 2024

App version Release date
2.0 July 2nd, 2024

New

Support Aztec barcode

You can now configure the scan engine settings in Knox Capture to recognize Aztec barcode, which is a type of 2D barcode. This increases the area of use cases that Knox Capture caters to.

Updates

Enhancements to home screen menu

The field descriptions are updated, the style of the screens is aligned with the latest Knox app style, which provides enhanced user guidance. On the support screen, you can see information such as the Samsung Knox Privacy Policy, About Knox Capture, and Quick help section. In the Quick help section, you can get resourceful links to Guides, FAQs, Tutorials, Release notes, and Contact us information.

Unlimited access to AR demo mode

To give you more opportunities to evaluate Knox Capture Augmented Reality (AR) features, the usage limits on AR demo mode are relaxed. Now, as long as you have a valid Knox Suite license, you can use the AR demo mode as often as you need. Even if you don’t have a valid Knox Suite license, you can still perform up to five scans.

AR template screens enhancements

The interface of the AR template screens is improved. The field names are updated, the style of the screens is aligned with the latest Knox app style.
