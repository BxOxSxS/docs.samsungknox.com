Glossary: N-S

Last updated March 13th, 2026

P

Pending

A status indicating an incomplete or awaiting action in Knox services. This includes:

  • An admin status where an invited admin must activate their Knox account after accepting an email invitation.

  • A device status where a reseller-uploaded device awaits admin approval before proceeding to enrollment.

  • A customer status where a Managed Service Provider (MSP) admin’s service addition request remains unresolved until the customer responds.

Applies to: All services, Samsung Care+ for Business


Personal profile

A dedicated Android partition that contains a user’s personal account, credentials, settings, storage, data, and apps. While admins cannot access personal profile content, they may apply device-wide policies which impact functionality.

Applies to: Knox Manage, Knox Mobile Enrollment, Knox Platform for Enterprise, Knox Service Plugin

Related terms: Work profile, Fully managed, Managed device


Permission

A right granted to a user, an admin, or an app to perform specific actions or access certain features related to Knox cloud services. It defines operational boundaries and helps preserve system integrity through granular control of functionality.

Applies to: All services

Related terms: Admin, Role


Profile

A profile is a collection or configuration of policies and settings applied to devices or users to control device behavior and functionality. Admins create and use profiles to deploy, enroll, and manage devices across their company.

Applies to: All services, Knox Authentication Manager

Related terms: Policy


Profile alias

A user-friendly identifier assigned by admins to an enrollment profile or a Knox Configure profile, and then shared with a reseller for easier recognition and management. Resellers can use these aliases to automate profile assignments to device uploads, without requiring manual action from their customer admins.

Applies to: Knox Configure, Knox Mobile Enrollment


Provisioned

A device status indicating that a device has been prepared for use in a company’s device management solution.

Applies to: Knox Mobile Enrollment

Related terms: Enrolled


Provisioning

In Knox Manage, a device status indicating that a device is in the process of being configured for management, but is yet to complete all provisioning steps.

Applies to: Knox Manage

Related terms: Enrolled


Policy

A configuration setting that controls specific device functionality or user permissions. Policies are typically bundled within profiles for deployment, enabling admins to enforce consistent management rules across their organization.

Applies to: All services

Related terms: Setting


Policy setting

A specific configuration value for a device, which defines the exact behavior or restriction applied within the broader policy.

Applies to: All services

Related terms: Policy, Setting


Push

The action of applying or sending profiles, policies, or settings from a Knox cloud service to a device, ensuring devices receive the latest updates or configurations.

Applies to: All services

R

Register

Refers to the following in the Knox Admin Portal:

  • The process of creating a user or admin account.

  • The process of adding an authorized reseller to the Knox Admin Portal. Once you register an authorized reseller, they can upload and provision devices on your behalf.

  • The process of adding devices from the common device list to a specific Knox cloud service. Admins must approve devices before registering them to a service. On registration, devices are shown in the service’s device list, allowing admins to manage them.

Applies to: All services

Related terms: Authorized reseller


Role-Based Access Control

A security framework that restricts admin permissions within Samsung Knox services. It assigns roles with specific access levels, ensuring admins only interact with features relevant to their responsibilities.

Applies to: All services, Samsung Care+ for Business

Related terms: Role


Role

A collection of permissions that determines what an admin can access and manage within Samsung Knox services.

Roles define the features and functions that an admin can use, such as viewing devices, managing licenses and settings, or inviting other admins.

Super admins can assign built-in roles or create custom roles with specific permissions that ensure appropriate access level based on user responsibilities. For more granular control, admins can be assigned one or more roles to tailor their access levels.

Applies to: All services

Related terms: Role-Based Access Control

S

Samsung account

A personal account that enables users to access Samsung’s consumer services, products, and applications. Samsung accounts provide authentication and user profile management across Samsung’s ecosystem, including device setup, app downloads, cloud storage, and online purchases.

This consumer-focused account type is distinct from Samsung account for Business, which is designed specifically for companies to manage and optimize their business operations, including authentication and access management.

Applies to: All services

Related terms: Samsung Account for Business


Samsung Account for Business

An identity and access management (IAM) platform designed to be at the core of IT operations, acting as a secure gateway to Samsung’s ecosystem of enterprise solutions — including Samsung Knox and Galaxy AI.

It is a dedicated work account for a company that offers centralized management and optimization of business operations, such as user account lifecycles, authentication, access control, and also enables scalable user onboarding and offboarding for reduced IT overhead.

With this work-exclusive account, companies can enforce enterprise-grade security protocols that maintain strict separation between corporate and personal data.

Following the 25.11 release, Samsung account for Business became the default account system for new Samsung Knox accounts, with existing Knox accounts getting the option to switch to Samsung account for Business from their personal Samsung accounts.

Applies to: All services

Related terms: Samsung account


Samsung Knox account

An account that allows you to register for and manage Knox cloud services.

Applies to: All services

Related terms: Samsung account, Samsung Account for Business


Server-based license

A license that provides access to Knox cloud and on-premises services. It’s managed centrally from a server console.

Applies to: All services


Service location

An automatically created entity in the Samsung account for Business console that links to a Knox company account, enabling centralized account management and access control for company devices.

Applies to: Fundamentals, Knox Admin Portal, Samsung account for Business


Service period

A Knox license’s operational timeframe during which devices can remain active and are managed. The period begins at license activation and grants continuous access to Knox features until license expiration.

For staggered licenses, the service period of a device may extend beyond license expiration depending on its support duration.

Applies to: All services

Related terms: Staggered license


Setting

A specific configuration option that controls how a device, application, or system behaves. This allows administrators to customize features, security policies, and functionality to meet enterprise needs.

Applies to: All services

Related terms: Policy


Setup edition

A Knox Configure staggered license modelthat enables one-time configuration and deployment of apps and settings to enrolled Samsung devices.

Applies to: Knox Configure

Related terms: Accessory edition, Staggered license


Setup profile

A Knox Configure profile applies a one-time configuration of apps and settings to Samsung devices with a setup edition license. Once applied, the configuration is retained until the device is factory reset, making it ideal for single-purpose devices.

Applies to: Knox Configure

Related terms: Setup Edition, Accessory profile, Dynamic profile


Staging user

A shared device profile applied during enrollment that prepares device for multi-user access. This profile establishes the sign-in interface and defines how the device operates when shared among multiple users.

Applies to: Knox Manage

Related terms: Kiosk mode


Staggered license

A Knox licensing model that provides continuous device support from enrollment through the service period, extending beyond the activation window. It calculates personalized support duration for each device based on its enrollment date within the activation period.

Applies to: Knox Configure, Knox Guard, Samsung Care+ for Business

Related terms: Activation period, Service period


Sub-admin

A restricted administrator role that helps manage Knox services with limited permissions. Invited by super admin or an admin with appropriate permissions, they can only access features specifically allowed by their assigned role.

Applies to: All services

Related terms: Admin, Super admin


Super admin

An administrator role with full permissions across Knox services. In the Knox Admin Portal, a super admin is the first registered enterprise user who registered for a Knox company account and can invite other admins. Knox Manage allows multiple super admins per Knox company account with complete system access.

Applies to: All services, Knox Manage

Related terms: Admin, Sub-admin
