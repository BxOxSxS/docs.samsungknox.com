Knox Guard 22.11 release notes

Last updated July 26th, 2023

Last server connection time on the device lock screen

You can now show the last time the device was connected to the Knox Guard server on its lock screen. This helps inform the device user that recent updates and actions pushed to the device might still be pending. To make use of this feature, the device must have an active network connection.

Note

To access this feature, you need Last Seen permission.

Support for secondary IMEI on dual-SIM devices

Previously, Knox Guard wouldn’t report the secondary IMEI (IMEI2) of dual-SIM devices.

Device search, device lists, and device detail pages on the console, as well as exported reports now support queries for a device’s secondary IMEI. If a dual-SIM device returns only one IMEI, the IMEI is denoted by either IMEI1 or IMEI2, depending on which SIM slot the card occupies on the device.
