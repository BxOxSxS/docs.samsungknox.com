Send relock timestamp and turn the relock reminder ON and OFF

Last updated July 26th, 2023

Notifications through this action are only applicable to PAYG customers.

Knox Guard manually sends a relock timestamp to the selected devices. Once that time is reached, devices will lock and display the customized message. Please note that you can also update the message without setting a new relock timestamp.

Send relock timestamp

To send a relock timestamp:

  1. Select the device from the checkbox on the left-hand side of the Devices page.

  2. Invoke the ACTIONS drop-down menu from the top right-hand side of the Devices page and select Manage device > Send relock timestamp.

  3. Define the 350 character maximum Message devices receive informing them how to remedy their delinquent subscription and unlock their device.

  4. Set relock timestamp to schedule a date and time to lock the selected devices.

  5. Select CONFIRM.

Send relock timestamp in bulk

A standardized CSV template is available to select devices on the Knox Guard console using their IMEI. The maximum number of devices that can be locked in bulk is 10,000.

To lock devices in bulk:

  1. Select BULK ACTIONS.

  2. Select the Download CSV template.

    For more information on how to create a properly formatted CSV file for bulk device acceptance, select Instructions for all bulk actions from the bottom, left-hand, side of the Bulk actions page.

  3. Select the SEND RELOCK TIMESTAMP from the Bulk actions page and add the CSV file on the Send relock timestamp page.

  4. Define the 350 character maximum Message devices receive informing them how to remedy their delinquent subscription and unlock their device.

  5. Set relock timestamp to schedule a date and time to lock the devices.

  6. Select SEND.

Knox Guard configures reminder messages to let the end users know that their devices will be locked soon. This feature can be applied to device(s) or turned off in the future.

Turn on relock reminder

To turn on the relock reminder:

  1. Select the device from the checkbox on the left-hand side of the Devices page.

  2. Select the ACTIONS drop-down menu from the Devices page and select Manage device > Turn on relock reminder.

Turn on relock reminder page showing the reminder duration and contact information options.

  1. Set the Reminder value to the number of days before the device relock timestamp will be reached.

  2. Define the 350 character maximum limit Message devices receive informing them how to remedy their delinquent subscription and unlock their device.

  3. Enter the Phone number of the contact resource locked device users call for restoring their locked device. Special characters can be used if necessary.

  4. Select TURN ON to apply the relock reminder.

Turn on relock reminder in bulk

A standardized CSV template is available to select devices on the Knox Guard console using their IMEIs. The maximum number of devices that can be selected in one CSV bulk upload is 10,000.

To turn on the reminder in bulk:

  1. Select BULK ACTIONS from the lower, left-hand, side of the page.

  2. Select the Download CSV template.

    For more information on how to create a properly formatted CSV file for bulk device acceptance, select Instructions for all bulk actions from the bottom, left-hand, side of the Bulk actions page.

  3. Select the TURN ON RELOCK REMINDER portion of the Bulk actions page and add the CSV file on the Turn on relock reminder page.

  4. Set the Reminder value to the number of days before the device relock timestamp will be reached.

  5. Define the 350 character maximum limit Message devices receive informing them how to remedy their delinquent subscription and unlock their device.

  6. (Optional) Select either Phone number to provide the support contact resource or Advanced support methods to provide the support app that locked device users can use to restore their locked device. Special characters can be used if necessary.

  7. Select TURN ON to apply the relock reminder.

Turn off relock reminder

To turn off relock reminder:

  1. Select the device from the checkbox on the left-hand side of the Devices page.

  2. Invoke the ACTIONS drop-down menu from the top right-hand side of the Devices page and select Manage device > Turn off relock reminder.

  3. Select CONFIRM to turn off the relock reminder for the selected devices.

Turn off relock reminder in bulk

A standardized CSV template is available to select devices on the Knox Guard console using their IMEIs. The maximum number of devices that can be selected in one CSV bulk upload is 10,000.

To turn off the reminder in bulk:

  1. Select BULK ACTIONS from the lower, left-hand, side of the page.

  2. Select the TURN OFF RELOCK REMINDER portion of the Bulk actions page and add the CSV file on the Turn off relock reminder page.

  3. Select the Download CSV template.

    For more information on how to create a properly formatted CSV file for bulk device acceptance, select Instructions for all bulk actions from the bottom, left-hand, side of the Bulk actions page.

    Turn off relock reminder page showing option to upload a CSV, XLS or XLSX file.

  4. Select CONFIRM to turn off relock reminders on the uploaded devices.
