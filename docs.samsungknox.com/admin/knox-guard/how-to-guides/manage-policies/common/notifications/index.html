Notifications

Last updated July 17th, 2025

You can send notifications to remind users about missing or overdue payments. Use the Notifications policy to set the default message and contact information shown when sending a notification to your devices.

To update this policy for select devices, see Send notifications.

Set notifications

To configure your notification settings:

  1. Navigate to the Policies page.

  2. Under MESSAGING, click NOTIFICATIONS.

  3. You can Initially show notifications as a full-screen message on user devices before they display on the notification panel. This makes notifications difficult to ignore.

    • (Optional) Click Preview full-screen message to view how the message displays on the device.

  4. (Optional) Enter a Phone number device users can call to make a payment and stop notifications. Phone numbers can only contain numbers and the following symbols: #, +, *.

  5. Enter the Notification 1, Notification 2, and Notification 3 message users receive from Knox Guard. In your message, you can:

    • Specify why the notification is sent to the user.
    • Use a different severity in the message tone, depending on how long the payment is overdue.
    • Enter a properly formatted URL (https://www…). When the user taps the URL hyperlink, the web page opens in the device’s browser. This can help make device payment queries as seamless as possible.

  6. If needed, click to ADD ANOTHER NOTIFICATION MESSAGE. You can add up to 10 messages.

  7. Click SAVE.
