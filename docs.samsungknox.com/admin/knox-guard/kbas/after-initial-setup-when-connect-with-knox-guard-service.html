After the initial device setup, when does it connect with the Knox Guard service?

Last updated July 26th, 2023

Categories:

Device

Overview

KG is enrolled once the setup wizard process is completed. No end user connection is required, KG client checks the server and enrolls the device automatically.

Additional information

  • For more information, see Knox Guard status flow.
