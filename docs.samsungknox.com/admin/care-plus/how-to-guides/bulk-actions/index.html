Bulk actions

Last updated April 9th, 2025

You can use a CSV file to update multiple devices in bulk on the Samsung Care+ for Business console. When creating a new CSV file:

  • Enter one device IMEI per line. Avoid commas to ensure device information is uploaded correctly to the portal.
  • When using Microsoft Excel, ensure that numeric device IDs don’t get converted into scientific notation format. You can correct it as follows:
    1. Select the rows containing the device IDs.
    2. On Home tab, select Fraction from the Number drop-down menu. Alternatively, select Number and set the Decimal places value to 0.

Upload devices

You can upload up to 10,000 devices at once:

  1. On the Devices page, click BULK ACTIONS > UPLOAD DEVICES.

  2. If this is your first bulk action, the Bulk action instructions page appears. Read the instructions, then click GOT IT.

  3. On the Upload devices screen, click BROWSE and select your CSV file.

  4. Click SUBMIT.

Devices you upload are automatically approved.

Bulk assign licenses

To bulk assign licenses:

  1. On the Devices page, click BULK ACTIONS > BULK ASSIGN LICENSES.

  2. If this is your first bulk action, the Bulk action instructions page appears. Read the instructions, then click GOT IT.

  3. Click BROWSE and select your CSV file.

  4. Select a license key to assign.

  5. Click SUBMIT.

Bulk delete devices

You can delete up to 10,000 devices at once:

  1. On the Devices page, click BULK ACTIONS > DELETE DEVICES.
  2. If this is your first bulk action, the Bulk action instructions page appears. Read the instructions, then click GOT IT.
  3. On the Delete devices screen, click BROWSE and select your CSV file.
  4. Click SUBMIT.
