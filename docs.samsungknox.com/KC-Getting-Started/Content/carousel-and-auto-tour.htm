Get started carousel and auto tour

This topic describes how to navigate and review the content of the KC get started carousel, and the KC console auto tour that displays immediately thereafter. These resources display only when a new user initially logs into KC, and are designed to assist new users in understanding central KC functions and better navigate the KC console.

Navigate the get started carousel and auto tour

Once your required Samsung and Knox Portal accounts are created and you first log into KC, a get started carousel displays describing the main tasks required to add devices, create profiles, and assign and deploy devices.

  1. A Welcome screen displays once initially logged into KC. Select the > on the right-side of the screen to proceed to the next screen in the get started carousel flow. Select SKIP TOUR to bypass the get started carousel and proceed to the KC console auto tour.

  1. The Add devices screen describes how the devices you have purchased from your reseller are automatically uploaded into your KC account on your behalf. You also have the option of using the Knox Deployment App (KDA), available from the Google Play store, to add devices automatically using a KC configured device profile.

For information on the Bluetooth, NFC and Wi-Fi enrollment options available to the KDA, go to: Knox Deployment App (KDA).

  1. Select the > on the right-side of the screen to proceed to the next screen in the get started carousel flow. If necessary, use select < to return to the get started Welcome screen. Select SKIP TOUR to bypass the get started carousel and proceed to the KC console auto tour.

  1. The Create profiles screen describes using KC to manage multiple profiles for use with different device settings.

Use the KC console to add new profiles, edit existing profiles and delete obsolete profiles as devices either initially enroll in KC or require update. Remember, before you can enroll devices into KC, you need to create a profile or edit an existing profile to meet enrollment requirements.

For information on using the KC console to create device profiles, go to Create profiles to assess whether a Setup or Dynamic edition KC profile best suits your deployment objectives.

For information on using the KDA to push a KC created profile to a device, go to: Knox Deployment App (KDA).

  1. Select the > on the right-side of the screen to proceed to the next screen in the get started carousel flow. If necessary, use select < to return to the get started Add devices screen. Select SKIP TOUR to bypass the get started carousel and proceed to the KC console auto tour.

  1. The Assign and deploy devices screen describes the KC trail license as an easy way to start deploying devices right away.
  1. Select the GOT IT button when you have completed reviewing the contents of the get started carousel. Once selected, you will not encounter any of the carousel screens again in subsequent logins.

  1. The display of the All set! screen signifies the end of the KC get started carousel. Select the NEXT button to transition to the KC auto tour.

The KC auto tour only displays once, and like the carousel is not displayed again in subsequent KC logins. The auto tour consists of a series of tooltips within the KC console that describe navigation tips, support resources, and account information.

The auto tour tooltips point to the following console navigation options:

  • Collapsible sidebar - Describes how to collapse the console sidebar navigation menu for console display optimization.
  • Bulk actions - Describes how to access the KC bulk actions button from the left navigation sidebar for access to bulks actions from one location.
  • Support and feedback - Describes the sidebar navigation buttons used to contact Samsung Support and provide feedback.
  • Account information - Describes how to access your user icon from the top of the console to access your customer ID and log out of the KC console.
  1. Once you have completed the KC get started carousel and auto tour, the KC console navigates to the Dashboard. For more information on the KC Dashboard, go to: Dashboard.
