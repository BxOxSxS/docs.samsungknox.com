App management

On this page

Knox Configure (KC) enables admins to upload select relevant Tizen based wearable device apps or custom watch face apps to utilize with custom device profiles. Additionally, you can also add apps that can be invoked after a hardware key is remapped.

After an app is uploaded to the Knox Configure (KC) console or to the Galaxy store, admins can manage device apps by adding apps to a profile.

Once an app is added to a profile, it pre-installs selected business apps onto KC provisioned devices.

Upload an application

  • Upload an app to the console
  • Add an app from the Galaxy Store

Upload app to the console

You can upload wearable apps into the Knox Configure console by following the steps below:

  1. Click UPLOAD APPLICATIONS.

  2. Click UPLOAD APPLICATION.

  3. Click BROWSE to select a valid APK, TPK or WGT file.
  4. Fill in the following information:
    • Application name — Enter a unique name for the in-house app.
    • Description — (optional) Provide additional details about the app. Maximum 245 characters.
  5. (Optional) If the file you're uploading is a watch face app, ensure you select This is a watch face application.
  6. Click UPLOAD.

After uploading the app, it will take approximately 3 days for the signing process. When the app is ready for deployment, you'll be notified by email and the app status will change to "Ready for deployment".

NOTE — An error message will appear if an unsupported file format is uploaded, or if the file exceeds a specific size, an error message. The supported file format is TPK or WGT.

Add an app from the Galaxy Store

You can add apps from the Galaxy Store by following the steps below:

  1. Click UPLOAD APPLICATIONS.

  2. Select ADD FROM GALAXY STORE.

  3. In the Add Wearable Application window, fill in the following information:
    • Package ID — For Tizen apps, a package ID uses the identifier "org.tizen.message" or "org.tizen.call". Ensure you enter the package ID, not the application ID. The Application name, App version, icon, and description may auto fill upon clicking SUBMIT. If a field remains blank, manually enter the remaining details.
    • Application name — Enter a unique name for the in-house app.
    • App version — Enter the app version.
    • Icon — (optional) Upload an app icon.
    • Description — (optional) Provide additional details about the app.
  4. Click UPLOAD.

After uploading the app, the app will be auto selected for deployment.

NOTE — For Tizen apps, a package ID uses the identifier "org.tizen.message" or "org.tizen.call". Ensure you enter the package ID, not the application ID.

Add a new app version

To update an app file with a new app version:

  1. On the left-hand navigation, go to Library.

  2. Select the WEARABLE APPS tab.

  3. Select the target app.

  4. Click ACTIONS > Add application version.

  5. Click BROWSE to select a valid APK, TPK or WGT file.
  6. (Optional) Enter a Description to provide additional details about the app. Maximum 245 characters.
  7. Click Upload.
NOTE — For more information on selecting applications for a Setup, Dynamic or Kiosk based profile, refer to the following: Setup edition, Dynamic edition or Kiosk mode

Upload an app to the Galaxy Store

Before proceeding, contact your Samsung admin to request permission to upload a B2B wearable app to the Samsung Galaxy Store.

To upload an in-house app to the Galaxy Store:

  1. Login to the Samsung Galaxy Store Seller Office.
  2. Click Add New Application.

  3. Select an application type to register: Select Galaxy Watch.
  4. Choose a default language.
  5. Click Next.
  6. Select Hidden radio button when submitting an in-house app.

  7. Follow the instructions to enter the App Information details, then click Save.
  8. Under Binary: Upload a binary by clicking Add Binary then click Save.
    • Note that up to 10 binaries can be uploaded. For additional information, read the Galaxy Watch App Type Guideline from Samsung Developers.

  9. Under the Country & Price tab, fill out the following details then click Save:
    • Price — Select Free.
    • Selling Starts — Click Selling starts on the day of approval or specify a date in advance.
    • US Export Laws — Select the check box to confirm that you have read and acknowledge the terms.
  10. Under Validation, upload the validation result for the hidden app, then click Save.

  11. Click Submit.

View app details

An app may have several versions uploaded, and to differentiate the latest version for potential device update, you can review app details to see more details about the app versions available.

To view app details:

  1. On the left hand navigation, go to Library.

  2. Select the WEARABLE APPS tab.

  3. Click on the target app to open the Application details window.
