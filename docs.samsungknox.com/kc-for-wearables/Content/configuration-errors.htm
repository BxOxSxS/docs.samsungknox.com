Error 51010: Error with <Configuration item>

Cause

There is a file or profile setting error.

Device message

Configuration Failed. An error occurred while configuring the device. Contact your IT admin or retry the configuration activity.

Log message

An error occurred with <API name> <application name> <content file name> in your profile. Check the profile configuration and retry.

Troubleshoot

  1. In the Knox Configure Portal, go to the Devices tab.
  2. Select the wearable device +experiencing the failure.
  3. Click Actions > View log. The log displays the error content (e.g. <API name: addContentShortcutIcon>, <content file name: abc.jpg>).
  4. Review the configuration file or setting experiencing the error. Ensure you have entered the correct file type or setting for that particular field.
  5. Factory reset the wearable device and try enrolling it again.

700101: 700100: Internal server error

Cause

The wearable device failed to connect to the Knox Configure agent due to a internal server error.

Device message

An error occurred while configuring the device. Contact your IT admin or retry.

Log message

Device internal error. If the problem persists, contact Samsung and provide a device log to assist with troubleshooting.

Troubleshoot

A system error occurred. Refresh the page and try again. If the problem persists, contact Samsung.
