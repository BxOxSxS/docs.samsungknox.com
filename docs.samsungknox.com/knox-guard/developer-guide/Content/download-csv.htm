Download devices as CSV

To download device information as a CSV:

  1. Navigate to the Devices screen.
  2. Select Download devices as CSV from the ACTIONS drop-down menu on the upper right side of the Devices screen.

The CSV will display the following device information:

  1. Refer to the IMEI for the device specific IMEI value.
  1. The SN column lists the SN used as device identifiers within the Knox Guard console.
  1. Refer to the MODEL column to review the list of models populating the Knox Guard console. Optionally use the column's filter function to refine which device models display.
  2. Refer to the STATUS column to assess whether each listed device is Activating, Active, Completing, Locked, Locking, Complete, Pending, Rejected, has a Reminder On, Resetting..., Starting reminder, Stopping reminder, Unlocking or Exchanging.
  3. Refer to the APP VERSION column to check apps installed with Knox Guard.
  1. Refer to the APPROVAL ID column to review the ID assigned when the device was approved for Knox Guard management.
  2. Refer to LAST SEEN column to display when a device was last detected by the Knox Guard server.
  3. The SERVICE ENDS column lists the date when the listed device is no longer supported by Knox Guard under the terms of its license.
  4. Refer to the CARRIER column to review and filter or search for various device carriers supporting each listed device.
  5. Refer to the ANDROID VERSION column to assess whether each listed device is using a current or legacy version of Android.
  6. Refer to the AGENT VERSION column to assess the current agent version of the device.
  7. Refer to UPLOADED column to check the upload status of the device.
  8. Refer to ACCEPTED column to check if the device has been accepted.
  9. Refer to ACTIVATED column to check if the device has been activated.
  10. Refer to the MODIFIED column to assess when the status of each listed device IMEI/SN was refreshed.
  11. Refer to the PREVIOUS IMEI/SNs to see the previous values the device was registered under.
  12. Refer to the SIM POLICY to see the current policies applied to the device.
