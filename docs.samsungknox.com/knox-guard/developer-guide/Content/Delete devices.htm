Delete devices

Consider permanently removing a device from Knox Guard when in a rejected or payment complete state. When a device is deleted from Knox Guard, the device is removed from the server and the Knox Guard client is uninstalled. When deleted, all of the device's information and history are removed.

Delete selected devices

To permanently delete a device from Knox Guard:

  1. Navigate to the Devices screen and select a Rejected or Payment Complete device checkbox.
  2. Select Delete device(s) from the ACTIONS drop-down menu on the upper right-hand side of the Devices screen.
NOTE - A selected device must be in a Pending, Rejected, Activating, Resetting, Completing or Payment Complete state for the Delete device(s) option to enable within the ACTIONS drop-down menu.

  1. Select the DELETE button to confirm the removal of the selected device.
  2. Select View device deletion log from the ACTIONS menu to review the devices removed from the Knox Guard console.

  1. Refer to the following deleted device log information:
  • IMEI/SN - Lists the serial number of each deleted device.
  • DEVICE MODEL - Lists the model of each deleted device. Consider filtering the display by model type to only list specific device models.
  • DATE DELETED - Use the filter options for this column to refine the device deletion date(s) as needed to capture device deletions historically.
  • CUSTOMER - Refer to this column to filter the deleted device display to just selected customers.
  • DELETED BY - Toggle the column display to list how device deleting resellers are displayed.
  1. Select EXPORT TO CSV to archive the deleted device log to a CSV file that can be saved outside of the Knox Guard console.

Delete devices in bulk

A standardized template is available to bulk delete devices using their IMEI. The maximum number of devices that can be bulk deleted is 10,000.

NOTE - A device must be in a Pending, Rejected, Activating, Resetting, Completing or Payment Complete state to delete.

To bulk delete devices:

  1. Select BULK ACTIONS from the lower, left-hand, side of the screen.
  1. Select the Download CSV template link from the top portion of the screen.
NOTE - As an additional bulk device deletion option, an Excel file (.xls or .xlsx) can be used if not wanting to use a CSV formatted file.
NOTE - If unsure of how to create a properly formatted file for bulk device deletions, select Instructions for all bulk actions from the bottom, left-hand, side of the Bulk actions screen.
  1. Select the DELETE DEVICES portion of the Bulk actions screen and add the CSV file into the Delete devices screen.

  1. Select the CONFIRM button to confirm the deletions of the device IDs within the uploaded CSV file.
NOTE - Once the device deletions are uploaded into Knox Guard the Device deletion log is updated accordingly.
