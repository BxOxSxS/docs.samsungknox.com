License management

On this page

A customer must purchase a license to use Knox Guard. As with other Knox Could Service (KCS) licenses, a customer can request a Knox Guard license from the sales team in their respective country. Once purchased, the customer enters their license into the Knox Guard console. A license is applied and activated automatically when a device is accepted into Knox Guard. When a license expires, the devices mapped to the expired license are automatically released from Knox Guard management control. Devices with an expired license cannot be registered to the Samsung Knox server.

NOTE - Each device has its own unique license period based on the time of its enrollment. Once a license is activated on a device, the license is irrevocable and cannot be transferred to a different device. Devices can be enrolled in Knox Guard up to 1 year after they are initially ordered. This one year enrollment window is known as the activation period. A license will expire if unused during the 1 year activation period.
NOTE - To avoid license utilization problems between trial and commercial licenses, always delete the trial license key from the tenant before adding production devices into Knox Guard.

Refer to the Licenses screen to enter, delete or acquire additional licenses when device activations reduce existing device license counts.

Manage Knox Guard licenses:

  1. Select Licenses from the left-hand navigation menu.

  1. Review the following Knox Guard license utilization values listed for each listed license:
  • LICENSE NAME - Displays the license name as a link that can be selected to review license details and, if necessary, update the license name. For more information, go to: License details.
  • TYPE - Lists whether a license is a Trial or Commercial license. Trial licenses are temporary and subject to expiration on the date listed within the Expiry column. Commercial licenses are for actual device deployments and have a much longer expiration period. Trial licenses can be deleted from the Knox Guard console when they are no longer required for device testing, or be converted to commercial licenses using the Samsung License Management (SLM) key generator. Always delete a trial license key before adding production devices. For more information, go to: Delete a license.
  • STATUS - Lists each license as either Expired, Active, or Registered (not yet activated). When a device is successfully enrolled into Knox Guard it moves into an Active state.
  • PURCHASED SEATS - Displays the maximum number of devices each listed license has paid to support. A Trial license supports 30 seats by default. Commercial licenses are negotiated point-of-sale for the number of license seats included.
  • REMAINING SEATS - Lists the total license count still available, minus assigned licenses. As devices successfully enroll and move to an Active state, the remaining license seat count decreases by 1.
  • ASSIGNED SEATS - Lists the number of devices assigned a license seat. The number listed also includes consumed license seats. When a device is accepted, the Assigned column count increases by 1.
  • ACTIVATED SEATS - Lists the number of devices that have been activated for each listed license. Activated seats are also included within the Assigned seats total.
  • ACTIVATION END DATE - Displays the expiration date after which licenses can no longer be assigned or activated on devices. License depletion emails are sent when the license count is down to 10% (out of total) remaining seats, then another email when at 5%, and a final email at 3%.
  1. Optionally use the Search field, to search and locate specific licenses if unable to initially locate them within the Licenses screen.

License details

To review the details of a specific selected license:

  1. Select Licenses from the left-hand navigation menu.
  2. Select the LICENSE NAME link of a specific license you wish to review in greater detail.

  1. The License Name is the only editable value within the License details screen. If needed, revise the name to something more applicable and select the SAVE button.
  2. The license Seats portion of the License details screen describes the following:
  • Purchased - Lists the number of license seats purchased on behalf of the selected license. Trial licenses support 30 devices by default, but can be transitioned to Commercial (perpetual) licenses as needed with the assistance of your reseller.
  • Remaining - Lists the total number of remaining license seats available out of the total number of license seats purchased for this selected license.
  • Assigned - When a device is accepted (enrolled) using this license, the Assigned column count increases by 1 to demonstrate the utilization of a license seat.
  • Activated - Lists the total number of devices that have been activated using this license.
  1. Displays the License Key used as a unique alphanumeric code to activate this specific license. Before you enter a Commercial key you should ensure the Trial key is deleted.
  2. Lists the Owner and email address of the device provider using Knox Guard to manage device subscriptions and payments using this license.
  3. Displays whether the selected license is a Trial or Commercial license. Trial licenses are temporary and subject to expiration upon the Activation End Date.Commercial licenses are for actual device deployments and have a much longer expiration period. Trial licenses can be deleted from the Knox Guard console when they are no longer required for device testing, or be converted to commercial licenses using the Samsung License Management (SLM) key generator.
  4. Lists the selected license Status as either Expired, Active (successfully enrolled into Knox Guard), or Registered (not yet activated).
  5. The Issue Date defines when the license was originally issued and available for allocation.
  6. The Activation Start Date defines the beginning date and time a license can be activated for device license seat allocation. The Activation Start Date, as opposed to the Issue date, begins the expiration countdown of a trial license's activation period.
  7. The Activation End Date lists end of the license seat activation period under the terms of this selected license. After this date, this license cannot be used to assign or activate additional devices. Devices that have already been activated with this license can remain active for the remainder of the Knox Guard 3 year service period. License depletion emails are sent when the license count is down to 10% (out of total) remaining seats, then another email when at 5%, and a final email at 3%.
  8. If necessary, select DELETE to permanently remove a Trial license when no longer required for device testing.

Get a license

To obtain a Trial (temporary) or Commercial (permanent) Knox Guard license:

  1. Select Licenses from the left-hand navigation menu.
  2. Select the GET A LICENSE button from the top, right-hand side, of the screen.

  1. Select GENERATE TRIAL LICENSE to create a license for the purposes of development and testing. Trial licenses are subject to expiration on the date listed within the DEVICES screen.
  2. Select CONTACT SALES to search for a Knox reseller in your specific geographic region that supports your intended Knox solutions.

Enter a license key

To enter a license into the Knox Guard console:

NOTE - When entering a Commercial license key and wanting future accepted devices to utilize this Commercial license, ensure your Trial license key is deleted if one is being utilized. Remaining Trial license seats will be consumed first, before the Commercial license seats are consumed.
  1. Select Licenses from the left-hand navigation menu.
  2. Select the ENTER LICENSE KEY button from the top, right-hand side, of the screen.

  1. Provide a relevant License name that will appear as a link within the Licenses screen. When selected, license information displays in detail. For more information, go to: License details.
  2. Enter the device's License key to include within the Device details screen. The License key is available from the device reseller as the device is enrolled in Knox.
  3. Select ADD to save the updates.

Out of licenses

There may be instances when you run out of devices licenses and are unable to activate additional devices into the Knox Guard console. When this occurs you will be prompted to increase your license count to accommodate the number of devices that exceed your current license capacity.

To add new licenses:

  1. If the Out of licenses screen displays when accepting devices, review the text stating how many licenses remain and how many licenses are required to accept the requested number of devices.
  2. Select Generate trial license to create a license for the purposes of development and testing. Trial licenses are subject to expiration on the date listed within the DEVICES screen. Only consider this option if the devices you are attempting to accept into Knox Guard are not for permanent commercial deployment.
  3. Refer to the Purchase commercial license portion of the screen for information on acquiring licenses for permanent commercial device deployments. Select Find your nearest reseller to locate a reseller in your area, as commercial licenses must be purchased from a Samsung reseller in your region.
  4. When you have received the requested license and key information, refer to the Enter license key field and enter the following:
  • Provide the License name to display as a license details link within the Licenses screen.
  • Enter the pending device's License key. The License key is available from the device reseller as the device is enrolled into Knox.
  1. Select ADD when completed to proceed with device acceptance using the updated license.

Delete a license

Trial licenses can be deleted and permanently removed from the Knox Guard console when they are no longer needed for device testing.

NOTE - To avoid license utilization problems between trial and commercial licenses, always delete the trial license key from the tenant before adding production devices.

To delete a trial license:

  1. Select a trial license from those available within the Licenses screen.
  2. Select the LICENSE NAME link of a listed trial license you wish to delete.
  3. Select the DELETE button from the bottom of the License details screen.

  1. Select DELETE to permanently remove the selected trial license. The deletion of the trial license is an irreversible action, so ensure the selected trial license is no longer needed for testing.
