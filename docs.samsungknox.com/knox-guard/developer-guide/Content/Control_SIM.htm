Control SIM

On this page

If necessary, control a selected device’s SIM functionality if the device requires SIM management separate from the default SIM control settings applied to other devices managed by Knox Guard. A Subscriber Identity Module (SIM) is an additional device security mechanism that assists service providers by locking a device when needed, applying usability restrictions for voice, text and data usage, and applying International roaming restrictions.

A mobile device’s SIM helps ensure the device is used the way the device’s service provider intends. When necessary, specific device subscriber SIM credentials can entered into Knox Guard to exclude them from the specific restrictions applied to other Knox Guard managed device SIMs.

SIM control can be disabled with Knox Guard when needed as mobile device ownership is changed or a new subscription plan applied.

Equally important to a secure SIM lock is a SIM unlock operation that does not compromise sensitive unlock code information. For information on unlocking a device's SIM module, go to: Unlock a device SIM.

NOTE - When an approved SIM is removed from a device a message displays indicating their device SIM cannot be detected or is from an unsupported service provider and requires replacement with an approved SIM.

For information on setting SIM control default values, go to: Manage defaults.

Enable SIM control

To enable SIM control on selected Knox Guard managed devices:

NOTE - SIM control can be applied to a device in any status state, except devices in a rejected state.

SIM control can be applied to a device in any status state, except devices in a Rejected state.

  1. Refer to the Devices screen and select a device checkbox.
  2. Select SIM Control from the ACTIONS drop-down menu on the upper right-hand side of the Devices screen.

  1. Refer to the MCC and MNC fields to assess if a default Mobility Country Code (MCC) and Mobile Network Code (MNC) has been applied to uniquely identify the selected device’s mobile network carrier for SIM control restriction exclusion. Each mobile device carrier has their own unique MCC and MNC pair. To revise the default MCC/MNC code assignment, go to: Manage defaults.

  2. Select the CONFIRM button to apply the SIM control settings to the device IDs specified within the CSV file.

  3. Set the following restrictions for an unlisted MCC/MNC:
    • Lock device - Select this option to lock a device's SIM functionality. This helps prevent using unlisted SIM cards. If you select 'Turn on network registration verification', it further validates that a deactivated SIM card is not being misused by comparing the defined MCC/MNC of the network operator.
    • Incoming/Outgoing calls - Select this option to restrict a selected device’s ability to make/recieve a voice call.
    • SMS/MMS/RCS - Select this option to restrict a selected device’s texting capability.
    • Data usage - Select this option to restrict a selected device’s ability to use data consumptive applications.
    • Apply the above restrictions when international roaming is in use – Select this option to restrict device voice, text and data usage when connecting to a mobile network in a country other than the device’s original deployment country. This option is applicable to non-whitelisted SIMs in a dual SIM device only.

  1. Refer to the Restricted action notification area to set the notification displayed when an unsupported SIM card has been inserted and usage restrictions have been applied.

    • Title - Define the required message title displayed to the device user attempting to insert an unsupported SIM in their device.
    • Message - Provide the required message content to the device user attempting to swap out their SIM for an unsupported one. The message should state that some functionality may be lost and to contact the subscriber for additional information.
  2. Select CONFIRM to commit the updated SIM control settings to the selected device. A notification message displays on the top portion of the Devices screen stating the selected device(s) will be SIM controlled.

Control device SIMs in bulk

A standardized CSV template is available to bulk control device SIMs using a device’s IMEI. The maximum number of devices that can be bulk SIM controlled is 10,000.

To bulk control device SIMs:

  1. Select BULK ACTIONS from the lower, left-hand, side of the screen.

  2. Select the Download CSV template link from the top portion of the screen.

NOTE - If unsure of how to create a properly formatted CSV file for bulk device SIM control, select Instructions for all bulk actions from the bottom, left-hand, side of the Bulk actions screen.

  1. Select the SIM CONTROL portion of the Bulk actions screen and add the CSV file into the SIM control screen.

  1. Set the following bulk device restrictions for an unlisted MCC/MNC:
  • Lock device - Select this option to lock a device's SIM functionality. This helps prevent using unlisted SIM cards. If you select 'Turn on network registration verification', it further validates that a deactivated SIM card is not being misused by comparing the defined MCC/MNC of the network operator.
  • Incoming/Outgoing calls - Select this option to restrict a selected device’s ability to make/recieve a voice call.
  • SMS/MMS/RCS - Select this option to restrict a selected device’s texting capability.
  • Data usage - Select this option to restrict a selected device’s ability to use data consumptive applications.
  • Apply the above restrictions when international roaming is in use – Select this option to restrict device voice, text and data usage when connecting to a mobile network in a country other than the device’s original deployment country. This option is applicable to non-whitelisted SIMs in a dual SIM device only.

  1. Other customer support options menu is also displayed on SIM lock screen but it is centrally configured in Default lock screen.

  2. Select the CONFIRM button to apply the SIM control settings to the device IDs specified within the CSV file.

Disable SIM control

Applied SIM controls can be disabled from selected devices as needed when SIM restrictions no longer apply.

To remove the control of a selected device’s SIM:

  1. Refer to the Devices screen and select a device checkbox that has control SIM applied.
  2. Select Disable SIM Control from the ACTIONS drop-down menu on the upper right-hand side of the Devices screen. The Disable SIM Control option is only enabled within the Actions menu if a SIM controlled device is selected.

  1. Select CONFIRM to disable SIM control from the selected devices. A notification message displays on the top portion of the Devices screen stating the selected device(s) have their SIM control disabled.

Bulk disable SIM control

A standardized CSV template is available to bulk remove SIM control using device IMEIs. The maximum number of devices that can have their SIM control bulk removed is 10,000.

To bulk remove device SIM controls:

  1. Select BULK ACTIONS from the lower, left-hand, side of the screen.

  2. Select the Download CSV template link from the top portion of the screen.

NOTE - If unsure how to create a properly formatted CSV file for bulk device SIM control, select Instructions for all bulk actions from the bottom, left-hand, side of the Bulk actions screen.

  1. Select the DISABLE SIM CONTROL portion of the Bulk actions screen and add the CSV file into the Disable SIM control screen.

  1. Select the CONFIRM button to disable the SIM control settings from the device IDs specified within the CSV file.
