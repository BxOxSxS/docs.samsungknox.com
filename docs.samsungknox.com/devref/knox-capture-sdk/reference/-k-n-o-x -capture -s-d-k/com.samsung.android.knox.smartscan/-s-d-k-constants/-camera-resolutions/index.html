enum CameraResolutions : Enum<SDKConstants.CameraResolutions>

This is the option value to setCameraResolution() in com.samsung.android.knox.smartscan.preview.data.PreviewConfig. RESOLUTION_1080P represent 19201080, RESOLUTION_UHD represent 38402160.

However, the real camera resolution may not be exactly as the value set here. CameraX will pick the best match if it is not supported
