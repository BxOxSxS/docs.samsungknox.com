public static final GraphicalView getCombinedXYChartView (Context context, XYMultipleSeriesDataset dataset, XYMultipleSeriesRenderer renderer, String[] types)

Creates a view that consists of the combination of multiple Charts like Line, Bar and Candle.

Parameters
context Contains the current context
dataset Contains all the data points required for plotting the graph (cannot be null)
renderer Contains all the styles and customizations to customize the graph (cannot be null)
types Set of chart types (cannot be null) to be shown on the view with values among Line,Bar,Candle or Scatter
Returns
  • Returns a GraphicalView that contains graphs of multiple types such as Line,Bar and Candle.
Throws
IllegalArgumentException Thrown if dataset is null or renderer is null or if number of items in the dataset is different than the number of chart types or if types has values apart from that defined for Line,Bar,Candle or Scatter chart
NullPointerException Thrown if ChartFactory is not initialized
