Class Overview

DeviceControlAgent
This class provides APIs to control remote device.
Control APIs include app launch, url launch and remote event dispatch.

To use this agent and APIs listed in class you should add permission to AndroidManifest:
1. Remote control (Event inject) : android.permission.sec.MDM_REMOTE_CONTROL
2. Application Managing : android.permission.sec.MDM_APP_MGMT
3. Shutdown & Reboot : android.permission.sec.MDM_SECURITY

public void remoteControl (int x, int y, int flag, String userId)

This is the API to control the remote device by touching.
Remote control of participant's device using point x and y values and touch event (remote dispatching touch event).
Position X and Y starts from 0 to 10000 default. You can change accuracy by calling initializeForRemoteControl (Not recommended).

Parameters
x X position which should be rate value.
y Y position which should be rate value.
flag Touch event flag (MotionEvent. down/up/move)
userId Target device owner's id

public void setPackageConvertMap (Map<String, String> convertMap)

If the user is using various devices, there is a need to Convert Map to Package Name in some applications.
ConvertMap is used when convert package name is changed.
Example, Package SNote is different so manager wants participants to launch it.
But the name of the SNote in participant's device is different.
To improve this situation, participant should add convert map :
convertMap = new HashMap();
convertMap.add(package name of new version, package name of current keep);

Parameters
convertMap Map of package name (expect to receive package name, having one)
