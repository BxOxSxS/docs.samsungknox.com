Class Overview

Agent Constant definition class
Each agent is added using its command list or minimum and maximum command values.
This class defines base command value of each agent for all the pre-defined agents.
For example,
Agent id of SessionManagingAgent is set 0, and the number of command count is 30.
So the range of command for SessionManagingAgent is from 0 to 30.

You can get Agent instance using Agent ID defined in this class :
SessionManagingAgent sessionAgent = (SessionManagingAgent) AgentManager.getInstance().getAgent(SESSION_MANAGING_AGENT);
OR
SessionManagingAgent sessionAgent = (SessionManagingAgent) AgentManager.getInstance().getAgent(SessionManagingAgent.class);
