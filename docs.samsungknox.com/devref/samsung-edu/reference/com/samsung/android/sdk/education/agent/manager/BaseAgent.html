Class Overview

BaseAgent class
This class is a basic class of Application logic which is managed by AgentManager.
BaseAgent also has ability to send command and data to others (Master or Participants) using sendData or sendDataUsingIp methods.
BaseAgent implements UserStatusChangedListener
1. Agent can know changes of user status
2. Each agent is registered to UserStatusNotifier when session is started

APIs in BaseAgent are :
1. created [protected / final] - this is called when BaseAgent is created and added to AgentManager
2. onInitialized [protected / abstract] - this is called after adding AgentManager
3. onSessionStarted [protected / final] - this is called when session is started successfully
4. onStarted [protected / abstract] - this is called at end of onSessionStarted to inform each agent
5. onSessionStopped [protected / final] - this is called when session is stopped
6. onStopped [protected / abstract] - this is called at end of onSessionStopped to inform each agent
7. onUpdate [protected / abstract] - this is called when data or message arrived if it is expected one
8. onDisconnected [protected / abstract] - this is called when device is disconnected from network
9. onReconnected [protected / abstract] - this is called when device is reconnected to network
10. isLocalIp [protected] - this API is to check given IP address is local IP or not
11. sendData [protected / final] - this API sends data to others
