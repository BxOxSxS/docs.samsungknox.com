Class Overview

Media Agent.

This class provides audio and video APIs

Supports :
1. The Teacher sends an audio data to Students while recording a voice.
- Students can play recorded data as soon as data is received
2. A teacher plays an audio file, he sends it to students
Students can play recorded audio data as soon as data is received.
3. After a teacher sends a video file to students, students can play it.

To use this agent and APIs listed in class you should add permission to AndroidManifest:
1. android.permission.RECORD_AUDIO
2. android.permission.WRITE_EXTERNAL_STORAGE
