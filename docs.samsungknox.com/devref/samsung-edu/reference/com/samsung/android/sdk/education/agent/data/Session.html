Class Overview

Session class
This class holds user list for those who creates or joins session.
This class also notifies changes of user state to maintain current state.

Session is created by calling Session constructor method.
Session(User myInfo, UserStateNotifier courseUpdatedNotifier) - For session master (creator)
Session(User myInfo, User manager, UserStateNotifier courseUpdatedNotifier) - For participants (joiner)
