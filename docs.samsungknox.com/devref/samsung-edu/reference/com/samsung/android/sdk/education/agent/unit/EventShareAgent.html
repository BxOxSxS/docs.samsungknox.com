Class Overview

EventShareAgent.
This is an agent of the Education SDK to show some event.
It includes notification and event sharing operations.

Supports :
1. Events : Get event from Calendar, Add and event and share with participants. To use this feature Account is already in device.
2. Notification : Send notification and it will be added in participant's NotificationBar.

To use this agent and APIs listed in class you should add permission to AndroidManifest:
1. For read and write events add permissions "android.permission.READ_CALENDAR" and "android.permission.WRITE_CALENDAR"
2. There is no specific permission about notification.
