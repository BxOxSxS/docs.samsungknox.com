XYMultipleSeriesRenderer
XYMultipleSeriesRenderer Provides different styles and customizations necessary for the developer to customize graph as per ones choice.The class is used for all the XY Charts. 

Class Overview

An abstract renderer to be extended by the multiple series classes.The class provides different styles and customizations necessary for the developer to customize graph as per ones choice.Both XY Charts and Round Chart can use the class.
