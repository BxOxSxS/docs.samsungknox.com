Class Overview

Received data from network.
When SDK receives data from the network, passes it to ReceivedData and transfer it to the respective agent.
This class contains method to retrieve useful info on the network data as follows :
1. command data for defining purpose
2. sender's IP address
3. Byte array data
4. JSONObject data if received data is json data, otherwise NULL
