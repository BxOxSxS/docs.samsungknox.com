Class Overview

DeviceManagingAgent

This class provides APIs to manage own device or remote devices.

Supports :
1. Install package (Now supporting local device - APKs in asset and sdcard)
2. Uninstall package (Both local and remote supported)
3. Request remote device information
4. Set the device settings such as wifi, gps, bluetooth, wallpaper, brightness, screen time out, and so on.

To use this agent and APIs listed in class you should add permission to AndroidManifest:
1. Application Managing : android.permission.sec.MDM_APP_MGMT
2. About Restriction : android.permission.sec.MDM_RESTRICTION, MDM_LOCATION, MDM_DATE_TIME, MDM_WIFI, MDM_BLUETOOTH
3. Change the settings value: android.permission.WRITE_SETTINGS
4. Set Wallpaper: android.permission.SET_WALLPAPER
