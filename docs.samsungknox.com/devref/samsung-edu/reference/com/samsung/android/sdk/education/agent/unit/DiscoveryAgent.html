Class Overview

DiscoveryAgent
This class provides APIs to broadcast and receive network information.

Supports :
DiscoveryAgent provides different types of discovery method
1. WIFI Multicast : Fast and Simple to announce and receive session information
2. BLE Bluetooth Low Energy : Device can advertise and scan session information through BLE
* To use this agent and APIs listed in class you should add permission to AndroidManifest:
1. WIFI Multicast : CHANGE_WIFI_MULTICAST_STATE
2. BLE Bluetooth Low Energy : android.permission.BLUETOOTH, android.permission.BLUETOOTH_ADMIN

public void stopReceivingSession ()

Stops receiving session announcement.
If participants do not want to receive session information, they can call this method
After calling this, thread which checking the validation of session information will be stopped.
And also DiscoveryAgent will not listen to Discovery information. This method only effects the Session information listener (Does not effect to Session Info Announcer : Master)
