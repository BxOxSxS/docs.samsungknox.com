Class Overview

DeviceLockAgent
This class provides APIs to locally or remotely lock devices and get the lock status of local and remote devices.

Supports :
1. Application Lock : Except on allowed application, other applications cannot be executed.
2. Screen Lock : The screen and key event is not dispatched, so after lock screen, user cannot control the device
3. StatusBar Lock : Notification and Statusbar can be locked


To use this agent and APIs listed in class you should add permission to AndroidManifest:
1. Application Lock : android.permission.sec.MDM_APP_MGMT
2. Statusbar Lock : android.permission.sec.MDM_KIOSK_MODE
