public int sendFile (List<String> files, List<String> idList, JSONObject metaData, FileShareListener fileLstnr)

This API is used for sending the list of files to the list of userIds.
The transaction created will be in queue. After finishing the pending transactions,
the current transaction will be executed.
Does not include senders id.

Parameters
files List of files to be sent
idList List of id's(User.userID) of the receivers to whom the file should be sent.
metaData Extra Meta data to be sent to the receiver :: can be null
fileLstnr Listener to receive the status of on going transaction
Returns
  • The transaction ID of the transaction for future operation purposes.
Throws
SecurityException android.permission.READ_EXTERNAL_STORAGE is required
