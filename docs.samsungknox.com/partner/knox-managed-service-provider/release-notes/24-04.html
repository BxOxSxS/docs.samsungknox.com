Knox MSP Program 24.04 release notes

Last updated April 3rd, 2024

New

Visibility into customer Knox service dashboard

As an MSP, you can now view a customer’s Knox service dashboard when accessing their services. Previously, you could not access the dashboard page when viewing a customer’s console.

Knox highlights dashboard module

A new Knox highlights module is now available on the dashboard. This panel provides featured content like the latest release notes, featured videos, and links to helpful resources to help you get the most out of the Knox products and services.

Updates

Duplicate permissions between portals

Several improvements were made to the Create role page. Now you can select whether you want to set permissions for both MSP Portal and service consoles at the same time, or whether you want to set permissions separately.

Additionally, now you can set all permissions to View only with a single click, making it easier to create support roles with fewer steps.

Seamless navigation between portals

When you go to your customer’s portal from the Knox MSP Portal, and then click the Back button in the top banner, the last page you had open in the Knox MSP Portal is restored.

Previously, any time you left the customer’s console, you’d always return to the Knox MSP Portal Customers page, even if you entered the customer console from a different Knox MSP Portal page.

Improvements to the What’s New dialog

The What’s New dialog that summarizes the changes of the latest release now includes an intuitive navigation menu to let you quickly read about the services that matter the most to you. You’ll now be able to see Knox service updates in addition to Knox MSP Portal updates. Previously, you could only view Knox MSP Portal updates in the What’s new dialogue.

Easily read and clear notifications

If you have an excessive number of unread notifications in the notification center, you can now easily read and clear up to 100 notifications at the same time. Simply click to display up to 100 notifications, then click anywhere outside dialog to clear the notifications from the list.

View customer’s Additional solutions page

As an MSP, you can now view the Additional solutions page when launching a customer’s Knox service console. Previously, this page was not available.

View Knox Manage devices in exported customer data

When you download your customer data as a CSV file, you’ll now see the total number of devices enrolled in Knox Manage in the TOTAL DEVICES (KNOX MANAGE) column, and the total number of devices enrolled in other Knox Suite services in the TOTAL DEVICES (EXCLUDING KNOX MANAGE) column. Previously, the CSV file did not include Knox Manage devices in the total device count.
