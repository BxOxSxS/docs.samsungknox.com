Knox MSP Program 23.09 release notes

Last updated September 6th, 2023

Dashboard enhancements

With this release, you can now view active license details from the Dashboard page of the portal. You can view the total number of active licenses, as well as sort licenses by end date or seat count to show licenses that are expiring soon or have a low seat count.

You can now also view the status of any open support tickets from the Dashboard.

Automatic license registration

With this release, licenses purchased by your Knox reseller using your MSP ID are now automatically registered and shown in the license table on the portal.

Support menu resource links

You can now access the admin guide, how-to videos, FAQs, KBAs, and release notes from the Support menu on the portal.

Email notification improvements

Notification emails sent to MSP admins have been rewritten to improve their formatting, provide clear calls to action, and create consistency with other Knox services.

Enterprise Edition license search

When generating a Knox Suite Enterprise Edition license for a customer, you can now search for the customer.
