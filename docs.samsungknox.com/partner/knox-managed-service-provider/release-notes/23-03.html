Knox MSP Program 23.03 release notes

Last updated April 25th, 2023

Update to supported Android versions

In accordance with our updated Android support plan, the minimum supported version for Android devices registered in Knox Mobile Enrollment, Knox Manage, Knox E-FOTA, Knox Configure, and Knox Deployment app have each rolled forward. To review the minimum version on a product-by-product basis, see Policy update on Knox supported versions.

Knox Mobile Enrollment link in Knox Suite license

A link to the Knox Mobile Enrollment console is now displayed along with other links to service consoles when you view a Knox Suite license that that is active for a Knox Mobile Enrollment advanced profile. If the customer isn’t using a Knox Mobile Enrollment advanced profile, this link will be greyed out and inactive.

Knox Asset Intelligence available as an MSP supported service

Knox Asset Intelligence is now available as an MSP-supported service alongside Knox Configure, Knox Mobile Enrollment, Knox E-FOTA, Knox Manage, Samsung Care+ for Business, and Knox Guard.

You can onboard new and existing Knox Asset Intelligence customers from the Knox MSP Portal and manage Knox Asset Intelligence on their behalf. Knox Asset Intelligence is also included as part of Knox Suite.

MSP admins can use both Knox Asset Intelligence individual licenses as well as Knox Suite licenses to access Knox Asset Intelligence. Knox Asset Intelligence licenses can be managed and assigned from the Licenses page on the Knox MSP Portal.

Removal of Knox Manage and Knox E-FOTA trial licenses

MSP admins can now no longer generate individual trial licenses for Knox Manage and Knox E-FOTA, and must instead generate a Knox Suite trial license.

Knox MSP Portal header UI improvements

The user interface of the Knox MSP Portal has been changed to remove redundant information and improve consistency. The options presented in the portal are now consistent with the actions that admins can perform. These UI changes don’t impact the functionality of the portal.

Trial license filtering

Previously, licenses in the Knox MSP Portal could only be filtered by the service to which they provide access. For MSPs with many customers, it was often difficult to differentiate between trial and commercial licenses.

You can now hide trial licenses in the license table to show only commercial licenses.
