Knox Partner Program 22.12 release notes

Last updated March 24th, 2023

Knox Partner Portal dashboard

The Knox Partner Portal now displays a common dashboard after sign-in, containing quick access links to onboarding and support resources, and marketing materials.

Tools provided in the Knox Partner Portal

Managed Service Provider (MSP) and Knox Deployment Program (KDP) partners can now access the Check License Key, Device Status, and Device Deactivation tools from the Knox Partner Portal menu. These tools were previously only available from the My Knox dashboard.

For details, see:

  • Check license keys
  • Check device status
  • Deactivate devices
