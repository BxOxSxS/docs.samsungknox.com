Reseller dashboard

Once you log in to the Samsung Reseller Portal, you will see a dashboard with the following information:

Your reseller ID — View the Reseller ID.
  • Reseller ID — The numeric reseller ID on the top of the dashboard is used as an identifier. Provide the reseller ID to your customers so they can add you as their designated reseller to help set up automated device upload.
    NOTE — Keep in mind, the reseller ID is not your customer ID.
TOP CUSTOMERS — View an ordered chart and numeric listing of top customers who has uploaded devices using Knox Configure.
  • VIEW ALL CUSTOMERS — Select the VIEW ALL CUSTOMERS link to quickly navigate to the Knox Reseller Portal Customers tab where all the customer device uploads can be reviewed.
TOP DEVICES — View an ordered chart and numeric listing of the device model names each top customer has uploaded using Knox Configure.
  • This information may be helpful in assessing the spectrum of device models utilizing Knox Configure.
RESELLER API KEY — View the current Reseller API key or click REGENERATE to create a new key to replace the previous key.
  • RESELLER API KEY — The listed reseller application programming interface (API) key allows other applications to access your reseller account without using your proprietary password.
SOLD-TO-CODE — View the sold-to-code (STC) assigned to a reseller account, or click ADD to add or edit an STC.
  • SOLD-TO-CODE — An STC is a code to manage Knox licenses. It's 10 characters in length and can include a hyphen (-). STCs are assigned to reseller accounts in the SLM.
CONTACT INFORMATION — View the Contact information.
  • CONTACT INFORMATION — Lists your reseller name and country of origin. This is the information your customers use to contact you and setup automatic device configuration uploads using Knox Configure.

Regenerate API key

The listed reseller application programming interface (API) key allows other applications to access your reseller account without using your proprietary password.

To create a new key, and obsolete an existing key:

  1. Log in to the Knox Reseller Portal.
  2. Go to Dashboard on the left-hand navigation.
  3. Under RESELLER API KEY: Click REGENERATE.

  4. Read the warning that regenerating an API key invalidates the existing key.

  5. Click GENERATE KEY.

Find a device

Find a device by entering one of the following on the search bar:

  • IMEI
  • MEID
  • Serial number

Manage a sold-to-code

To add sold-to-codes (STCs) to a reseller account:

  1. Sign in to the Knox Reseller Portal.
  2. On the dashboard, in the SOLD-TO-CODE(S) card, click ADD.
    NOTE — If you have previously added STCs to your account, click Manage to add or remove STCs.

  3. On the Add or manage sold-to-code(s) page, click ADD AND MANAGE SOLD-TO-CODE.

  4. Enter the code, provide an alias name, and click SAVE.

  5. To add more STCs, click ADD ANOTHER SOLD-TO-CODE. You can register up to a maximum of 20 STCs.
