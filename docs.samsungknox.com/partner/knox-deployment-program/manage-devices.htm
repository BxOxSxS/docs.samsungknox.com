Manage devices

Knox Configure requires resellers to submit device information on behalf of their customers before profiles can be pushed to enrolled devices.

Through the reseller portal, resellers are able to review devices uploaded to a customer, upload devices, and delete devices.

Upload devices

To upload devices to the Reseller Portal, you first need to prepare a CSV file that contains the IMEIs for the devices to be uploaded. Devices for upload can't already be registered in an inventory that you manage through Knox services.

Prepare a CSV file

To create a list of devices for a single customer, prepare a CSV file as follows:

  • Don't add any header rows or lines.

  • Enter one device IMEI per line.

  • If you're uploading any dual-SIM devices, enter either of the two associated IMEIs.

  • Avoid commas, to ensure device information is uploaded correctly into the portal.

  • If you use a spreadsheet app like Microsoft Excel to create your CSV file, to prevent numeric device IDs from being converted into scientific notation, set the cell format to Number, and set Decimal places to 0.

To create a list of devices in bulk for one or more customers, prepare a CSV file as follows:

(You can also download this CSV template file. See Upload devices for one or more customers for details.

  • 1st value or column — Customer ID. Make sure each row has only one customer ID.

  • 2nd value or column — Device IMEI/serial number. Make sure each row has only one IMEI/serial number. If you're uploading any dual-SIM devices, enter either of the two associated IMEIs.

  • 3rd value or column (optional) — Order number. The order number must contain only alphanumeric characters.

  • 4th value or column (optional) — Order date. The order date must be in YYYYDDMM format.

  • Avoid commas (except for the separator commas if you're creating the CSV file in a text editor), to ensure device information is uploaded correctly into the portal.

  • You can upload a maximum of 10,000 devices per CSV file.

  • If you use a spreadsheet app like Microsoft Excel to create your CSV file, to prevent numeric device IDs from being converted into scientific notation, set the cell format to Number, and set Decimal places to 0.

Upload devices for a specific customer

  1. Sign in to the Knox Reseller Portal.

  2. Go to Customers.

  3. Under Company name / Internal reference, select the target customer/MSP link.

    If you have MSP and reseller accounts that share the same MSP ID, other resellers can upload devices to your MSP account.

  4. Click the UPLOADS tab.

  5. Click UPLOAD DEVICES.

  6. Enter the following information:

    • Device CSV file — Click BROWSE to upload the CSV file you prepared.
    • Sold-to-code (optional) — This field is available if you've added sold-to-codes in the Reseller Portal. If you have added multiple STCs, select the correct STC from the list.
    • Order number (optional) — Enter the order number for the listed customer's devices.
    • Order date (optional) — Click the calendar to select the date when the customer placed the order.
    • Tags (optional) — Search for devices after they have been uploaded to Knox Configure.

  7. Click Upload to submit the devices for the selected customer.

Upload devices for one or more customers

  1. On the Reseller Portal, click BULK ACTIONS.
  2. Click UPLOAD DEVICES.
  3. Click BROWSE to upload the CSV file you prepared.
    TIP — Or, download a CSV file template that contains the required column headers, fill it, and then upload it.
  4. Click SUBMIT to submit the devices.

Download device list or uploads list

After you've uploaded devices to the Reseller Portal, you can download a list of devices or uploads as a CSV file.

  1. On the Knox Reseller Portal, go to Customers.
  2. Under Company name / Internal reference, select the target customer/MSP link.
  3. On the UPLOADS tab, click ACTIONS.
  4. Choose either Download uploads as CSV or Download devices as CSV. When you download devices, both IMEIs of any dual-SIM devices are included in the exported CSV file.

View devices

  1. Log in to the Knox Reseller Portal.
  2. Select the Customers tab.

  3. Under Company name / Internal reference, select the target customer link.

  4. Select the DEVICES tab.

  5. Sort submitted devices by any of the column headers, or search for devices.

    When searching for devices by IMEI, dual-SIM devices can be found by either of their IMEIs. The device list shows only IMEI1 (the primary IMEI), but when you hover over a dual-SIM device, a tooltip shows both associated IMEIs.

    6. NOTE — Rejected devices will generate a CSV error event message, but will no longer be displayed within the Devices table.

Device details

On the Devices tab, click the device that you want to get additional details for.

For dual-SIM devices, both IMEIs are shown under IMEI/MEID.

Other fields are described in the Upload devices section.

Wi-Fi: phone MAC address is the latest addition to these features. This address can be used to restore the device in the database and allowlist the device Wi-Fi address.

Delete devices

  • Delete a single device
  • Delete multiple devices for a specific customer by selecting them
  • Delete multiple devices for a specific customer using a CSV file
  • Bulk delete multiple devices for one or more customers using a CSV file

Delete a single device

  1. Sign in to the Knox Reseller Portal.
  2. Go to Customers.

  3. Under Company name / Internal reference, select the target customer link.

  4. Select the UPLOADS tab.

  5. Select a device.

  6. Click ACTIONS > Delete device upload.

Delete multiple devices for a specific customer by selecting them

To delete one or more devices by selecting them from a customer's device list:

  1. Log in to the Knox Reseller Portal.
  2. Select Customers from the left-hand navigation menu.
  3. Under Company name / Internal reference, select the target customer link.
  4. Select the DEVICES tab.
  5. Select those target devices you intend to delete.
  6. From the ACTIONS list, select Delete devices.

Delete multiple devices for a specific customer using a CSV file

To delete a large number of devices from the Reseller Portal, you first need to prepare a CSV file that contains the IMEIs for the devices to be deleted:

See the Upload devices for a specific customer section for details about the required CSV file format.

To delete one or more devices using a CSV file upload:

  1. Sign in to the Knox Reseller Portal.
  2. Go to Customers.
  3. Under Company name / Internal reference, select the target customer link.
  4. Click the DEVICES tab.
  5. From the ACTIONS list, select Delete devices via CSV.
  6. Click BROWSE to upload the CSV file you prepared.
  7. Click SUBMIT.

Bulk delete multiple devices for one or more customers using a CSV file

To delete a large number of devices from the Knox Reseller Portal, you first need to prepare a CSV file that contains the customer IDs and IMEIs for the devices to be deleted.

See the Upload devices for one or more customers section for details about the required CSV file format.

To bulk delete devices:

  1. Sign in to the Knox Reseller Portal.
  2. Click BULK ACTIONS in the sidebar.
  3. Click DELETE DEVICES.
  4. Click BROWSE to upload the CSV file you prepared.
  5. Click SUBMIT.
