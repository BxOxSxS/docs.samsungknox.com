View the dashboard

Last updated September 10th, 2025

The Dashboard page allows you to view and manage important information at a glance.

The dashboard displays the following information:

Component Description
Your Reseller ID Your unique identifier. Provide your reseller ID to your customers so they can add you as their designated reseller and set up device auto-approval.
TOP CUSTOMERS

An ordered chart of the customers with the most uploaded devices.

Click VIEW ALL CUSTOMERS to go to the Customers page, where you can review all customer device uploads.
TOP DEVICES An ordered chart of the most uploaded device models.
KNOX CLOUD API

Launches the Knox Cloud API portal.

Visit the Knox Cloud API portal to generate authentication keys and reseller client IDs. For details, see the Knox cloud services authentication and Knox Deployment Program developer documentation.

This option is available if you have the **Access Knox Cloud APIs portal** permission. If you require API access, please contact support.
SOLD-TO-CODE(S)

The number of sold-to-codes (STCs) assigned to your reseller account.

An STC is a 7-10 character code used to manage Knox licenses. STCs are assigned to reseller accounts in the Samsung Sales Portal. To add and manage STCs, see Manage sold-to-codes.
CONTACT INFORMATION Your reseller name, address, and country of origin. Your customers can use this information to contact you.

Find a device

You can find a device by entering one of the following device identifiers in the search bar:

  • IMEI
  • MEID
  • Serial number

Manage sold-to-codes

You can add or manage sold-to-codes (STCs) for a reseller account on the Dashboard page. An STC is a unique identifier assigned to Knox license resellers in the Samsung Sales Portal, where the Samsung License Management (SLM) system is integrated. Select an STC to designate a Knox license reseller to renew any licenses related to the Knox Suite - Enterprise Plan.

You must specify an STC when uploading Enterprise Edition devices.

To add or manage sold-to-codes (STCs):

  1. On the SOLD-TO-CODE(S) card, click ADD. If you’ve previously added STCs to your account, click MANAGE. The Add or manage sold-to-code(s) page opens.

  2. Click ADD AND MANAGE SOLD-TO-CODE. If you’ve previously added STCs to your account, click ADD ANOTHER SOLD-TO-CODE. The Add or edit a Sold-to-code dialog displays.

  3. Enter the code and an alias name, then click SAVE.

    You can register up to a maximum of 20 STCs. To assign an STC to a specific set of devices, see Upload devices for a specific customer.

  4. To edit or delete an existing STC, click the icon for the STC you wish to manage.
