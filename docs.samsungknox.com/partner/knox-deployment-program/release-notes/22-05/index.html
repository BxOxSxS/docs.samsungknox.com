Knox Deployment Program 22.05 release notes

Last updated April 25th, 2023

Sold-to-code management for resellers

Previously, resellers could only register one sold-to-code (STC) per reseller account on the Knox Reseller Portal.

With Knox Deployment Program 22.05, resellers with more than one STC can register multiple codes to an account. Furthermore, when uploading devices, resellers can now select the correct STC from the registered list of STCs.
