About this White Paper

This White Paper provides an overview of the Samsung Knox ML Model Protection, discusses the security challenges faced by the AI and ML industries and Samsung Knox as a viable solution to mitigate them.

This document is designed for security professionals, IT managers, AI/ML developers, and others evaluating Knox ML Model Protection as a solution. For additional information, go to the Samsung Knox website.
