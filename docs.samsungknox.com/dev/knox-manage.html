Knox Manage

Last updated August 5th, 2025

Introduction

Welcome to the Knox Manage REST API reference. You can integrate using these APIs to:

  • Administer users, groups, and devices
  • Manage access, authenticate users, and send device commands
  • Push profiles and apply policy changes
  • And much more.

Refer to relevant sections later on this page for details.

Access to the API
