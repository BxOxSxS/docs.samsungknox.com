Knox Manage 24.09 release notes

Last updated January 8th, 2025

Ability to send notifications to Android and iOS devices

Knox Manage 24.09 introduces the Notification to device operation that you can use to send notifications to Android and iOS devices. Additionally, for Android devices, you can also specify whether the notification is to be displayed as a push notification or as a pop-up.

Enhancement to the Get Device Location operation

With Knox Manage 24.09, a new parameter — stdFormatUpdates — is added to the Get Device Location operation. The stdFormatUpdates parameter returns date and time information in the ISO format.

Deprecations

The following APIs have been deprecated and will be removed in a future release:

  • /user/createDevice
  • /user/deleteDevice
  • /user/activateUser
  • /user/deactivateUser

Learn more about admin features

To learn about features available to admins in this release, see Knox Manage 24.09 release notes in the admin guide.
