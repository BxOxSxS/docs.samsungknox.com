Knox Manage 24.04 release notes

Last updated April 3rd, 2024

New requestEnrollment operation

With Knox Manage 24.04, you can now use the new POST user/requestEnrollment operation to email the enrollment guide to a device user.

Enhancement to the Read Audit Log operation

Knox Manage 24.04 provides two new parameters - startLogTime and endLogTime – for the Read Audit Log operation to record time in the format hh:mm. The hours and minutes data collected using the parameters is available in the audit logs.

Learn more about admin features

To learn about features available to admins in this release, see Knox Manage 24.04 release notes in the admin guide.
