Can I use Knox E-FOTA to set the highest firmware version allowed on multiple devices?

Yes.

  • EMM developers — Use an Android app to call the Knox SDK API method RestrictionPolicy.setAllowedFOTAVersion. This sets the highest firmware version that can be installed on a device.
  • IT admins — Use an EMM console to centrally push a firmware version policy to a specific group of devices. The Android app on those devices would then call the setAllowedFOTAVersion method to set the highest firmware version allowed.
