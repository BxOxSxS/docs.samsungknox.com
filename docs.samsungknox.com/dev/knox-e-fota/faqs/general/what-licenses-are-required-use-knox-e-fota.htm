What licenses are required to use Knox E-FOTA?

To use the Samsung Knox E-FOTA, you need the following:

  1. Knox Platform for Enteprise (KPE) license key - An Android app on a device needs to activate this license to use the Knox SDK, which is used to set the highest firmware version allowed. Developers registered with Knox Partner Program can get a license from the Knox Partner Program portal.
  2. REST API Key (client ID, client secret) - REST API calls to the Samsung Knox E-FOTA REST API service need to provide this key to identify the source of calls. Knox Partner Program partners can request a key through the Knox Partner Program portal.
