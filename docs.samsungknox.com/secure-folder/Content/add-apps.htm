Add apps to Secure Folder

On this page
There are a variety of ways for you to add apps to Secure Folder.

Download from app store (Google play / Galaxy apps)

  1. Tap Add apps.
  2. Choose an app store.
    • NOTE – you may be required to sign into your Google or Samsung account again.
  3. Search for an app and install it.

Add apps from phone

  1. Tap Add apps.
  2. Select apps from the list, or search apps through search window.

  3. Tap Add. Selected apps are shown in Secure Folder.

Quick menu on Home / Apps

  1. Press & hold on an app from the Home screen or App screen → Select Add to Secure Folder.
  2. Unlock Secure Folder (User authentication).
    • If Secure Folder is unlocked, apps will be added to Secure Folder instantly.
  3. The app you selected is shown in Secure Folder.

NOTE – Some apps may not operate properly inside Secure Folder due to technical limitations.
NOTE – Some apps may require you to set up and configure them again.

Add shortcuts to Home screen

You can add direct app shortcuts to Secure Folder apps on your personal home screen. This allows you to launch Secure Folder apps quickly.

  1. Tap Edit apps from the Secure Folder home screen.
  2. Select an apps → Tap Add to shortcut to Home.
  3. The added shortcut is shown on the Home screen.
  4. Now you can launch Secure Folder apps quickly with the shortcut on the Home screen.

NOTE – You can also add app shortcuts by pressing & holding on the app inside Secure Folder. Tap Add shortcut to Home in the menu that pops up. If you tap Select multiple items, you can add multiple shortcuts at once.
NOTE – Secure Folder apps can be distinguished by the small blue ‘badge’ in the right corner of the icon.
